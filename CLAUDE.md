## Ключевые формулы
- Капитализация = кол-во акций × цена акции
- EV = капитализация + чистый долг
- EBITDA = операционная прибыль + амортизация
- FCF = CFO − Capex
- FCFE = FCF − проценты − погашенный долг + выпущенный долг
- P/E = капитализация / чистая прибыль
- EV/EBITDA = стоимость бизнеса / EBITDA
- ND/EBITDA = чистый долг / EBITDA (>3x = высокая нагрузка)
- ROE = чистая прибыль / капитал
- DCF: PV = FVn / (1+r)^n
- Терминальная стоимость: TV = FCF × (1+g) / (r−g)

## Правила ответов

- Сравнивай мультипликаторы с отраслевыми средними
- При анализе: сначала качественные критерии, потом количественные

- В конце аналитики: краткий вывод + риски
