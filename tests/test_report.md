## Топ-10 проблем (по критичности)

### КРИТИЧЕСКИЕ (блокируют ответ)

**1. Неполные данные в companies_data.json — отсутствует shares_count**
- Затронуто: 5+ компаний (Сбер, Татнефть, Башнефть, Сургут, БСП)
- Влияние: невозможно рассчитать EPS, BVPS, Graham Number, целевую цену DCF
- Решение: добавить `shares_count` и `price` для ВСЕХ компаний
- Файл: `knowledge/companies_data.json`

**2. Компании вне базы: Сегежа, Лента, Позитив, Астра, HeadHunter, OZON**
- Затронуто: 6+ вопросов по секторам, рискам, портфелям
- Влияние: бот не может дать числовой анализ
- Решение: расширить JSON до 40-50 компаний
- Файл: `knowledge/companies_data.json`

**3. Только 1 временной период — нет динамики**
- Затронуто: ВСЕ вопросы по анализу (CLAUDE.md требует "ДИНАМИКА всегда")
- Влияние: нельзя показать рост/падение выручки, маржи, ROE за 3-5 лет
- Решение: добавить данные за 2 года (текущий + прошлый) или поля `revenue_growth`, `ebitda_growth`
- Файл: `knowledge/companies_data.json`

### ВАЖНЫЕ (снижают качество)

**4. Данные устарели (last_updated: 2024, бот работает в 03.2026)**
- Затронуто: все вопросы с конкретными цифрами
- Влияние: мультипликаторы, цены, дивиденды могут быть неактуальны
- Решение: обновить данные до 2025-2026; добавить в CLAUDE.md правило "предупреждать о дате данных"
- Файлы: `knowledge/companies_data.json`, `CLAUDE.md`

**5. Нет EBIT, Invested Capital, Current Assets/Liabilities в JSON**
- Затронуто: Magic Formula (Гринблатт), Z-Score (Альтман), NCAV (Грэм), Current Ratio
- Влияние: ключевые скоринговые модели нельзя рассчитать точно
- Решение: расширить JSON полями `ebit`, `invested_capital`, `current_assets`, `current_liabilities`, `total_liabilities`
- Файл: `knowledge/companies_data.json`

**6. Нет EPS growth / momentum данных**
- Затронуто: PEG (Линч), Momentum Score, факторное инвестирование
- Влияние: стратегии роста и momentum неприменимы количественно
- Решение: добавить `eps_growth_3y`, `eps_growth_5y`, `price_return_12m`
- Файл: `knowledge/companies_data.json`

**7. Скилл dcf.md не обрабатывает edge cases**
- Затронуто: DCF для банков, убыточных, циклических компаний
- Влияние: бот может дать некорректную модель или зависнуть
- Решение: добавить ветвления: "если банк → Residual Income/DDM", "если FCF < 0 → NAV/SOTP", "если циклическая → нормализованный FCF"
- Файл: `.claude/skills/dcf.md`

### ЖЕЛАТЕЛЬНЫЕ (улучшат UX)

**8. Portfolio скилл не всегда спрашивает профиль инвестора**
- Затронуто: вопросы "какие акции купить?", "собери портфель"
- Влияние: ответ без уточнения горизонта/риска может не подойти пользователю
- Решение: добавить в portfolio.md обязательное уточнение перед рекомендацией
- Файл: `.claude/skills/portfolio.md`

**9. Нет knowledge файла по ИИС и налогообложению**
- Затронуто: вопросы про налоги, ИИС, ЛДВ
- Влияние: поверхностные ответы о налоговых льготах
- Решение: создать `knowledge/tax_and_iis.md`
- Файл: новый

**10. Нет исторических данных по кризисам (2008, 2014, 2022)**
- Затронуто: сравнение с историческими периодами
- Влияние: ответы основаны на общих знаниях, а не на базе
- Решение: создать `knowledge/market_history_ru.md`
- Файл: новый
