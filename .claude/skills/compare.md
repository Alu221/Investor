# Сравнение компаний по мультипликаторам

## Когда использовать
Пользователь просит сравнить 2-3 компании, или спрашивает "что лучше — {A} или {B}?"

## Алгоритм

### 0. Проверка доступности (ОБЯЗАТЕЛЬНО)

**Если компании НЕТ в @knowledge/companies_data.json** (криптовалюты, иностранные акции, не торгуемые на MOEX):
→ Вежливо объясни: "Моя специализация — российский рынок (MOEX). У меня нет данных по {компания}."
→ Предложи альтернативу: "Могу сравнить с другими компаниями из базы (~45 компаний MOEX)"
→ Если одна из двух компаний есть в базе — дай анализ по ней

**Если компания — банк** (Сбер, ВТБ, Тинькофф, Совкомбанк, БСП):
→ Не использовать EV/EBITDA, FCF (нерелевантны для банков)
→ Вместо этого: P/E, P/BV, ROE, дивидендная доходность, NI
→ Ключевая комбинация для банков: высокий ROE + низкий P/BV = недооценка

**Если показатель = null** (NI, FCF, ROE у убыточных):
→ Ставить "—" в таблице и пояснить: "данные отсутствуют (компания убыточна)"

### 1. Загрузить данные
Из @knowledge/companies_data.json загрузи данные обеих компаний.

### 2. Таблица сравнения
Построй таблицу:

| Показатель | {Компания A} | {Компания B} | Лучше |
|-----------|-------------|-------------|-------|
| Капитализация | | | |
| P/E | | | ← ниже лучше |
| P/S | | | ← ниже лучше |
| EV/EBITDA | | | ← ниже лучше |
| ND/EBITDA | | | ← ниже лучше |
| P/BV | | | ← зависит от сектора |
| ROE | | | ← выше лучше |
| Маржинальность EBITDA | | | ← выше лучше |
| FCF | | | ← выше лучше |
| Дивидендная доходность | | | ← выше лучше |
| Группа (2026) | | | |

### 3. Анализ различий
- Почему одна компания дешевле/дороже другой
- Какие риски учтены в цене
- Качественные различия (менеджмент, регуляторика)

### 4. Вывод
- Какая компания привлекательнее и почему
- Для какого инвестора подходит каждая (дивидендный / ростовой / стоимостной)
- Если есть target_price — указать таргет и потенциал для обеих

### 5. Дисклеймер
> Данный анализ не является индивидуальной инвестиционной рекомендацией.
