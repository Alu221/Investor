# Сравнение компаний по мультипликаторам

## Когда использовать
Пользователь просит сравнить 2-3 компании, или спрашивает "что лучше — {A} или {B}?"

## Алгоритм

### 1. Загрузить данные
Из @knowledge/companies_data.json загрузи данные обеих компаний.

### 2. Таблица сравнения
Построй таблицу:

| Показатель | {Компания A} | {Компания B} | Лучше |
|-----------|-------------|-------------|-------|
| Капитализация | | | |
| P/E | | | ← ниже лучше |
| P/S | | | ← ниже лучше |
| EV/EBITDA | | | ← ниже лучше |
| ND/EBITDA | | | ← ниже лучше |
| P/BV | | | ← зависит от сектора |
| ROE | | | ← выше лучше |
| Маржинальность EBITDA | | | ← выше лучше |
| FCF | | | ← выше лучше |
| Дивидендная доходность | | | ← выше лучше |
| Группа (2026) | | | |

### 3. Анализ различий
- Почему одна компания дешевле/дороже другой
- Какие риски учтены в цене
- Качественные различия (менеджмент, регуляторика)

### 4. Вывод
- Какая компания привлекательнее и почему
- Для какого инвестора подходит каждая (дивидендный / ростовой / стоимостной)

### 5. Дисклеймер
> Данный анализ не является индивидуальной инвестиционной рекомендацией.
