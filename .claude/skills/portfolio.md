# Конструктор портфеля

## Когда использовать
Пользователь просит "собери портфель", "куда вложить N рублей", "какие акции купить"

## Алгоритм

### 1. Определи профиль инвестора
Спроси (если не указано):
- **Горизонт**: 1-2 года / 3-5 лет / 10+ лет
- **Риск**: консервативный / умеренный / агрессивный
- **Цель**: доход (дивиденды) / рост капитала / сохранение
- **Сумма**: для расчёта кол-ва лотов

### 2. Выбери модель портфеля

**Консервативный (доход):**
- 60% дивидендные акции (доходность > 10%)
- 25% value с moat (P/E < 8, ROE > 15%)
- 15% ОФЗ или кэш

**Умеренный (баланс):**
- 40% дивидендные
- 30% value/quality
- 20% ростовые (PEG < 1)
- 10% ОФЗ/кэш

**Агрессивный (рост):**
- 40% quality growth
- 30% Magic Formula (ROIC + EY)
- 20% momentum
- 10% кэш для новых идей

### 3. Отбор компаний
Из @knowledge/companies_data.json + @knowledge/russian_stocks_2026.md:
- Применить фильтры по модели
- Проверить по @knowledge/universal_checklist.md
- Max 5-10% портфеля на одну компанию
- Диверсификация по 3+ секторам
