# Конструктор портфеля

## Когда использовать
Пользователь просит "собери портфель", "куда вложить N рублей", "какие акции купить"

## Алгоритм

### 1. Определи профиль инвестора (ОБЯЗАТЕЛЬНО)
**ВСЕГДА спрашивай перед составлением портфеля, если пользователь не указал:**
- **Горизонт**: 1-2 года / 3-5 лет / 10+ лет
- **Риск**: консервативный / умеренный / агрессивный
- **Цель**: доход (дивиденды) / рост капитала / сохранение
- **Сумма**: для расчёта кол-ва лотов

⚠️ НЕ давай рекомендации без профиля! Если пользователь говорит "просто собери портфель" — задай хотя бы 2 вопроса: горизонт и риск.

### 2. Выбери модель портфеля

**Консервативный (доход):**
- 60% дивидендные акции (доходность > 10%)
- 25% value с moat (P/E < 8, ROE > 15%)
- 15% ОФЗ или кэш

**Умеренный (баланс):**
- 40% дивидендные
- 30% value/quality
- 20% ростовые (PEG < 1)
- 10% ОФЗ/кэш

**Агрессивный (рост):**
- 40% quality growth
- 30% Magic Formula (ROIC + EY)
- 20% momentum
- 10% кэш для новых идей

### 3. Отбор компаний
Из @knowledge/companies_data.json + @knowledge/russian_stocks_2026.md:

**ФИЛЬТР ПО ГРУППАМ (ОБЯЗАТЕЛЬНО):**
- Консервативный портфель → ТОЛЬКО группы 1-2
- Умеренный → группы 1-3
- Агрессивный → группы 1-4 (группа 4 — не более 10% портфеля)
- Группа 5 (ПИК, Камаз, Аэрофлот, Самолёт, Мечел) → НИКОГДА не включать в портфель

**ДОПОЛНИТЕЛЬНЫЕ ФИЛЬТРЫ:**
- ND/EBITDA > 3x → НЕ включать (кроме спец. случаев с обоснованием)
- net_income = null (убыточные) → НЕ включать в консервативный/умеренный
- equity < 0 (МТС) → НЕ включать в консервативный
- P/E > 25 (Астра, Позитив, OZON, Яндекс) → только в агрессивный, не более 5-7% каждая

**МИНИ-АНАЛИЗ КАЖДОЙ КОМПАНИИ ПЕРЕД ВКЛЮЧЕНИЕМ (ОБЯЗАТЕЛЬНО):**
Для каждого кандидата в портфель выполни быструю проверку:
1. WebSearch "[компания] новости последние" — нет ли свежего негатива (делистинг, убытки, санкции, смена менеджмента)
2. get_price(ticker) — актуальная цена, не упала ли резко (> 20% за месяц = красный флаг)
3. Проверить из companies_data.json:
   - FCF > 0? (отрицательный = не включать в консервативный)
   - ND/EBITDA < 3? (выше = не включать)
   - Дивиденды стабильны? (для дивидендного портфеля)
   - ROE > 10%? (ниже = неэффективна)

Если хотя бы 1 красный флаг — ИСКЛЮЧИТЬ и заменить другой компанией. Объяснить почему исключена.

- Проверить по @knowledge/universal_checklist.md
- Max 5-10% портфеля на одну компанию
- Диверсификация по 3+ секторам

### 4. Результат
Таблица (если указана сумма — рассчитай лоты из lot_size и lot_cost в JSON):
| Компания | Сектор | Вес | Лотов | Сумма (руб) | Почему | Ключевой риск |
|----------|--------|-----|-------|-------------|--------|---------------|

Пример расчёта лотов:
```
Сумма портфеля: 500 000 руб
Сбер (вес 15%): 75 000 руб / lot_cost 3202 = 23 лота = 230 акций × 320 руб = 73 600 руб
Лукойл (вес 15%): 75 000 / 5807 = 12 лотов (= 12 акций) × 5807 = 69 684 руб
```

### 5. План ребалансировки
- Частота: ежеквартально
- Триггер: отклонение > 5% от целевого веса
- Правила выхода: ND/EBITDA > 3x, потеря moat, делистинг
