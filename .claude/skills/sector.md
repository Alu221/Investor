# Анализ сектора

## Когда использовать
Пользователь спрашивает "что с нефтянкой?", "какой сектор сейчас лучший?", "сравни банки"

## Алгоритм

### 1. Макро-контекст сектора
- Как текущая ставка ЦБ влияет на этот сектор?
- Как курс рубля/нефти влияет?
- Есть ли санкционные риски для сектора?
- Регуляторные изменения (НДПИ, пошлины, тарифы)?

### 2. Компании сектора
Из @knowledge/companies_data.json выбрать все компании сектора и построить таблицу:

| Компания | P/E | EV/EBITDA | ROE | ND/EBITDA | Дивиденды | Группа |
|----------|-----|-----------|-----|-----------|-----------|--------|

### 3. Средние по сектору
Рассчитать медианные мультипликаторы сектора для сравнения.

### 4. Рейтинг внутри сектора
Ранжировать компании по привлекательности:
1. **Лучшая** — почему (1-2 предложения)
2. **Вторая** — почему
3. ...
n. **Худшая** — почему

### 5. Драйверы и риски сектора
- Что может ускорить рост сектора (каталисты)?
- Что может обрушить сектор (угрозы)?

### 6. Вывод
- Сектор привлекателен / нейтрален / опасен в текущих условиях
- Топ-1-2 компании для покупки
- Компании, которых стоит избегать

### 7. Дисклеймер
> Данный анализ не является индивидуальной инвестиционной рекомендацией.
