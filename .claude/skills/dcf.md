# Расчет DCF-модели

## Когда использовать
Пользователь просит рассчитать DCF, справедливую стоимость, или целевую цену акции.

## Алгоритм

### 0. Проверка применимости (ОБЯЗАТЕЛЬНО перед расчётом)

**Если компания — БАНК** (Сбер, ВТБ, Тинькофф, Совкомбанк, БСП):
→ DCF по FCF неприменим (у банков нет классического FCF)
→ Использовать **Residual Income** или **DDM (модель Гордона)** из @knowledge/advanced_valuation.md
→ Для DDM: P = D1 / (r - g), где D1 = ожидаемый дивиденд, r = CAPM, g = рост дивидендов

**Если EBITDA < 0 или FCF < 0 (убыточная компания)** (Русал, ВК):
→ DCF невозможен — нечего дисконтировать
→ Использовать **NAV** (чистые активы), **SOTP** (для холдингов) или **P/BV** из @knowledge/advanced_valuation.md
→ Объяснить пользователю почему DCF не работает и предложить альтернативу

**Если компания ЦИКЛИЧЕСКАЯ** (металлургия, нефтегаз при волатильных ценах):
→ Использовать **нормализованный FCF** = средний FCF за 3-5 лет (а не текущий пиковый/дно)
→ Указать: "Текущий FCF может быть на пике/дне цикла, используем нормализованное значение"

**Если shares_count отсутствует в JSON:**
→ Рассчитать: shares_count = mcap / price (оба поля должны быть)
→ Если оба отсутствуют — предупредить пользователя и дать EV без цены на акцию

### 1. Входные данные
Запроси у пользователя или возьми из @knowledge/companies_data.json:
- FCF текущий (или FCFE) — для циклических: нормализованный
- Темп роста FCF на прогнозный период (обычно 5 лет)
- Терминальный темп роста g (обычно 2-4% для РФ)
- Ставка дисконтирования r (обычно 15-20% для РФ, ≈ ключевая ставка ЦБ + премия за риск)
- Кол-во акций (shares_count из JSON или рассчитать mcap/price)

### 2. Прогноз FCF (5 лет)
```
Год 1: FCF₀ × (1 + темп роста)
Год 2: FCF₁ × (1 + темп роста)
...
Год 5: FCF₄ × (1 + темп роста)
```

### 3. Терминальная стоимость
```
TV = FCF₅ × (1 + g) / (r - g)
```

### 4. Дисконтирование
```
PV(FCFₙ) = FCFₙ / (1 + r)ⁿ
PV(TV) = TV / (1 + r)⁵
```

### 5. Справедливая стоимость
```
EV = Σ PV(FCF) + PV(TV)
Equity Value = EV - чистый долг
Цена акции = Equity Value / кол-во акций
```

### 6. Анализ чувствительности
Построй таблицу: как меняется целевая цена при разных ставках дисконтирования (±2%) и темпах роста (±1%).

### 7. Вывод
- Целевая цена акции
- Потенциал роста/падения от текущей цены (%)
- Ключевые допущения и их обоснование
- Что может пойти не так

### 8. Дисклеймер
> DCF-модель основана на прогнозах и допущениях. Не является индивидуальной инвестиционной рекомендацией.
