# Анализ загруженного финансового отчета

## Когда использовать
Пользователь загружает Excel/PDF с финансовой отчетностью компании и просит проанализировать.

## Алгоритм

### 1. Идентификация документа
- Определи компанию и период отчетности
- Определи тип: МСФО или РСБУ
- Определи вид: P&L, Баланс, ОДДС, или комбинированный

### 2. Извлечение ключевых данных

**Из P&L (отчет о прибылях и убытках):**
- Выручка
- Валовая прибыль и маржинальность
- Операционная прибыль (EBIT)
- EBITDA (= EBIT + амортизация)
- Чистая прибыль

**Из Баланса:**
- Активы (оборотные / внеоборотные)
- Обязательства (краткосрочные / долгосрочные)
- Капитал
- Чистый долг = долг − кэш

**Из ОДДС:**
- Операционный денежный поток (CFO)
- Инвестиционный (CFI) → Capex
- Финансовый (CFF) → дивиденды, долг
- FCF = CFO − Capex

### 3. Расчет мультипликаторов
Если есть текущая цена акции — рассчитай:
- P/E, P/S, P/BV, EV/EBITDA, ND/EBITDA, ROE, P/FCF

### 4. Динамика
Если данные за несколько периодов — покажи динамику (рост/падение в %).

### 5. Вывод
- Сильные стороны отчета
- Слабые стороны / тревожные сигналы
- Сравнение с отраслевыми аналогами (из базы данных)

### 6. Дисклеймер
> Данный анализ не является индивидуальной инвестиционной рекомендацией.
