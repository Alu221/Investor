# Стресс-тест портфеля / компании / сектора

## Когда использовать
Пользователь спрашивает "что будет если...", "stress-test", "сценарий", "что если нефть упадёт"

## Алгоритм

### 1. Определи тип стресс-теста

**Макро-шок:**
- Нефть Brent падает до $40-50 → влияние на нефтяников
- Ставка ЦБ растёт до 25% → влияние на закредитованных
- Рубль слабеет до 120/$ → влияние на экспортёров vs импортёров
- Новые санкции → влияние по секторам из @knowledge/macro_russia.md

**Секторный шок:**
- Отмена льготной ипотеки → девелоперы
- Падение цен на металлы → металлурги
- Технологический прорыв → IT-сектор

**Корпоративный шок:**
- Отмена дивидендов → пересчёт доходности
- Рост долга → пересчёт ND/EBITDA, Z-Score
- Потеря ключевого рынка → Газпром-сценарий

### 2. Рассчитай влияние

**Для макро-шока (нефть, ставка, рубль):**
Используй таблицу чувствительности из @knowledge/macro_russia.md:
```
Нефть -30% → выручка нефтяников -25-35%, EBITDA -40-50%
Ставка +4% → WACC +3-4%, DCF-оценка -15-25%
Рубль -20% → прибыль экспортёров +15-25% (в рублях)
```

**Для каждой компании из @knowledge/companies_data.json:**
```
1. Текущие показатели: EBITDA, FCF, ND/EBITDA, div_yield
2. Стресс-показатели: EBITDA_stress = EBITDA × (1 - шок%)
3. Новый ND/EBITDA = net_debt / EBITDA_stress
4. Если ND/EBITDA_stress > 3x → КРАСНАЯ ЗОНА
5. FCF_stress = FCF × (1 - шок%)
6. Если FCF_stress < 0 → дивиденды под угрозой
```

### 3. Корреляции
Используй @knowledge/correlation_matrix.md:
- Нефтяники коррелируют с нефтью (+0.80)
- Девелоперы коррелируют со ставкой ЦБ (-0.70)
- Золото — хедж от кризиса (+0.50 к инфляции)
- Сургут-п — хедж от девальвации (-0.80 к USDRUB)

### 4. Результат

Таблица стресс-теста:
| Компания | Текущий ND/EBITDA | Стресс ND/EBITDA | Текущий FCF | Стресс FCF | Дивиденды | Риск |
|----------|-------------------|------------------|-------------|------------|-----------|------|

Цветовая разметка:
- 🟢 Выживет без проблем (ND/EBITDA < 2, FCF > 0)
- 🟡 Под давлением (ND/EBITDA 2-4, FCF ±)
- 🔴 Критично (ND/EBITDA > 4, FCF < 0, дивиденды под угрозой)

### 5. Рекомендации по хеджированию
- Какие позиции сократить (наиболее уязвимые)
- Какие добавить (хеджи: золото, ОФЗ, экспортёры)
- Каш-позиция (сколько держать в кэше для просадок)

### 6. Исторический контекст
Из @knowledge/market_history_ru.md — что происходило при аналогичных шоках:
- 2008: нефть $147→$36, рынок -74%
- 2014: санкции + нефть, рынок -20%
- 2022: геополитика, рынок -45%

### 7. Дисклеймер
> Стресс-тест основан на приблизительных моделях и допущениях. Реальное влияние может отличаться. Не является индивидуальной инвестиционной рекомендацией.
