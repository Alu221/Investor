# Анализ рисков компании

## Когда использовать
Пользователь спрашивает "какие риски у X?", "безопасно ли покупать?", "что может пойти не так?"

## Алгоритм

### 1. Финансовые риски
- **Долговая нагрузка**: ND/EBITDA (> 3x = опасно при растущих ставках)
- **Ликвидность**: Current Ratio (< 1 = не хватает на текущие обязательства)
- **Качество прибыли**: OCF vs Net Income (OCF < NI = красный флаг)
- **Altman Z-Score**: < 1.8 = высокий риск банкротства
- **Beneish M-Score**: > -2.22 = вероятны манипуляции с отчётностью

### 2. Операционные риски
- **Зависимость от одного продукта/клиента**
- **Сезонность бизнеса**
- **Конкуренция** (есть ли moat?)
- **Качество менеджмента** (государственное vs частное)

### 3. Макрориски
- **Санкционный**: экспортёр? попадает под ограничения?
- **Валютный**: выручка в $ при расходах в ₽ (или наоборот?)
- **Процентный**: высокий долг + растущая ставка ЦБ = удар по прибыли
- **Сырьевой**: зависимость от цены нефти/газа/металлов/удобрений
- **Регуляторный**: НДПИ, экспортные пошлины, тарифное регулирование

### 4. Политические риски
- **Госвладение**: может изменить дивидендную политику приказом
- **Санкции**: компания/владелец под санкциями?
- **Геополитика**: конфликты, торговые войны

### 5. Рыночные риски
- **Ликвидность торгов**: низкий объём = широкий спред
- **Делистинг-риск**: уход с биржи
- **Блокировка иностранных активов**

### 7. Вывод
- Суммарный уровень риска: НИЗКИЙ / СРЕДНИЙ / ВЫСОКИЙ
- Главные 2-3 риска, за которыми следить
- При каких условиях нужно выходить из позиции

### 8. Дисклеймер
> Данный анализ не является индивидуальной инвестиционной рекомендацией.
