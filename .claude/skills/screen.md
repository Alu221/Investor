# Скрининг акций по критериям

## Когда использовать
Пользователь просит найти акции по критериям: "недооцененные", "дивидендные", "с низким долгом", "ростовые".

## Алгоритм

### 1. Определи критерии
Типичные фильтры:

**Стоимостные (Value):**
- P/E < 7
- P/BV < 1
- EV/EBITDA < 5

**Дивидендные (Income):**
- Дивидендная доходность > 10%
- Стабильная история выплат

**Низкий долг:**
- ND/EBITDA < 1 (или отрицательный = нет долга)

**Эффективные:**
- ROE > 20%
- Маржинальность EBITDA > 30%

**Ростовые (Growth):**
- Рост выручки > 15% г/г
- P/S < 2

### 2. Фильтрация
Загрузи @knowledge/companies_data.json и отфильтруй компании по заданным критериям.

### 3. Результат
Выведи таблицу подходящих компаний:

| Компания | Сектор | P/E | EV/EBITDA | ROE | Дивиденды | Группа |
|----------|--------|-----|-----------|-----|-----------|--------|

### 4. Краткий комментарий
По каждой компании — 1 предложение: почему подходит, главный риск.

### 5. Дисклеймер
> Данный скрининг не является индивидуальной инвестиционной рекомендацией.
