# Полный фундаментальный анализ компании

## Когда использовать
Пользователь просит проанализировать компанию, оценить акцию, или спрашивает "что думаешь про {компания}?"

## Алгоритм

### 0. Проверка данных (ОБЯЗАТЕЛЬНО)

**Если компании НЕТ в @knowledge/companies_data.json:**
→ Сообщи: "У меня нет данных по {компания} в базе. Могу дать общий качественный анализ без цифр, или проанализировать похожую компанию из базы."
→ Если компания упоминается в @knowledge/russian_stocks_2026.md — использовать группировку и качественную оценку

**Обработка null-значений:**
→ Если net_income = null — не считать P/E, ROE, EPS. Указать: "Компания убыточна, P/E нерелевантен"
→ Если fcf < 0 — указать как красный флаг, объяснить причину (инвестфаза? проблемы?)
→ Если equity < 0 — ОБЯЗАТЕЛЬНО отметить как критический красный флаг

**Для банков:** не использовать EV/EBITDA, FCF. Ключевые метрики: P/E, P/BV, ROE. Дополнительно из @knowledge/balance_sheet.md: Н1 (>12%), NPL (<5%), NIM (3-5%), CIR (<40%). Правило: высокий ROE + низкий P/BV = недооценка.

### 1. Качественный анализ
- Определи сектор и основной бизнес компании
- Кто акционеры и менеджмент (государственная/частная)
- Регуляторные риски (санкции, пошлины, льготы)
- Конкурентные преимущества
- Ключевые драйверы роста и риски

### 2. Количественный анализ
Загрузи данные из @knowledge/companies_data.json и рассчитай:

**Оценка стоимости:**
- Капитализация (Mcap) = кол-во акций × цена
- EV = Mcap + чистый долг
- P/E, P/S, P/BV, P/FCF — сравни с отраслевыми средними
- EV/EBITDA — основной мультипликатор
- ND/EBITDA — долговая нагрузка (>3x = опасно)

**Эффективность:**
- ROE — возврат на капитал
- Маржинальность по EBITDA
- Динамика выручки и прибыли (год к году)

**Денежные потоки:**
- FCF — свободный денежный поток
- FCFE — для акционеров
- Дивидендная доходность

### 3. Сравнение с аналогами
Сравни мультипликаторы с компаниями того же сектора из базы данных.

### 4. Вывод
- Оценка: недооценена / справедливо оценена / переоценена
- Если есть target_price — указать таргет и потенциал
- Ключевые плюсы (2-3 пункта)
- Ключевые риски (2-3 пункта)
- К какой группе относится (из @knowledge/russian_stocks_2026.md)
- Если data старше 6 мес — предупредить о дате

### 5. Дисклеймер
> Данный анализ не является индивидуальной инвестиционной рекомендацией.
