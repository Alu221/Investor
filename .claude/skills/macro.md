# Макроэкономический анализ

## Когда использовать
Пользователь спрашивает "что сейчас с рынком?", "куда вкладывать при такой ставке?", "как санкции влияют?"

## Алгоритм

### 1. Текущий макро-контекст
Проанализируй (из @knowledge/macro_russia.md):
- Ключевая ставка ЦБ РФ и тренд (растёт/падает/стабильна)
- Инфляция (ускоряется/замедляется)
- Курс USDRUB и тренд
- Цена нефти Brent
- Санкционный фон (новые ограничения?)

### 2. Влияние на секторы
| Фактор | Выигрывают | Проигрывают |
|--------|-----------|-------------|
| ↑ ставка ЦБ | Банки (спред), облигации | Строительство, ростовые |
| ↑ нефть | Нефтегаз, бюджет | Авиа, транспорт |
| ↓ рубль | Экспортёры (нефть, металлы) | Импортёры, IT (долларовые расходы) |
| Санкции | Импортозаменители | Экспортёры под ограничениями |

### 3. Тактика
- При ставке > 15%: фокус на дивидендные акции + ОФЗ-ПК (флоатеры)
- При ставке 8-15%: баланс дивидендных и ростовых
- При ставке < 8%: агрессивно в ростовые + длинные ОФЗ
- На пике ставок: покупать длинные ОФЗ (при снижении цена растёт)

### 4. Рекомендация по распределению
Предложи конкретное распределение портфеля для текущих условий:
- % в акциях (какие секторы)
- % в облигациях (короткие/длинные, ОФЗ/корпоративные)
- % в кэше/золоте

### 5. Дисклеймер
> Данный анализ не является индивидуальной инвестиционной рекомендацией.
