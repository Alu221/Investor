# Продвинутые методы оценки стоимости

## WACC — средневзвешенная стоимость капитала

### Формула
```
WACC = (E/V × Re) + (D/V × Rd × (1 - t))

E = рыночная стоимость акционерного капитала
D = рыночная стоимость долга
V = E + D (общая стоимость)
Re = стоимость акционерного капитала (из CAPM)
Rd = стоимость долга (средняя ставка по кредитам)
t = ставка налога на прибыль (20% в РФ)
```

### Расчёт для российской компании
```
Пример: Лукойл
E = 4 500 млрд руб (капитализация)
D = 300 млрд руб (чистый долг)
V = 4 800 млрд руб
Re = 18% (из CAPM с учётом странового риска)
Rd = 14% (средняя ставка по облигациям)
t = 20%

WACC = (4500/4800 × 18%) + (300/4800 × 14% × 0.8)
WACC = 16.88% + 0.70% = 17.58%
```

### Особенности для РФ
- Страновой риск: добавлять 3–5% к безрисковой ставке
- Актуальную ставку ЦБ и доходность ОФЗ — через WebSearch
- Для компаний без долга: WACC = Re (стоимость собственного капитала)
- Высокий WACC = высокая ставка дисконтирования = ниже справедливая стоимость

## CAPM — модель оценки капитальных активов

### Формула
```
Re = Rf + β × (Rm - Rf) + CRP

Re = ожидаемая доходность акционерного капитала
Rf = безрисковая ставка (доходность ОФЗ 10 лет)
β = бета-коэффициент (чувствительность к рынку)
Rm - Rf = премия за рыночный риск (5–7% для РФ)
CRP = премия за страновой риск (3–5% для РФ в 2025–2026)
```

### Бета-коэффициенты типичных секторов РФ
| Сектор | β примерный | Комментарий |
|--------|-------------|-------------|
| Нефть и газ | 0.8–1.1 | Зависит от нефти, умеренная β |
| Банки | 1.1–1.4 | Выше рынка, цикличны |
| Металлы | 1.2–1.5 | Высокая цикличность |
| Телеком | 0.5–0.8 | Защитный сектор |
| Электроэнергетика | 0.3–0.6 | Самая низкая β |
| IT/Технологии | 1.3–1.8 | Высокая волатильность |
| Ритейл | 0.7–1.0 | Умеренная |

### Пример расчёта
```
Rf = 15% (доходность ОФЗ 26243 на март 2026)
β = 1.1 (Сбербанк)
Rm - Rf = 6%
CRP = 4%

Re = 15% + 1.1 × 6% + 4% = 25.6%
```

## Gordon Growth Model (модель постоянного роста)

### Формула
```
P = D1 / (r - g)

P = справедливая цена акции
D1 = ожидаемый дивиденд в следующем году
r = требуемая доходность (Re из CAPM)
g = темп роста дивидендов (бессрочно)
```

### Когда применять
- Зрелые компании со стабильными дивидендами
- g < r (обязательное условие)
- История дивидендов 5+ лет
- Подходит для: Лукойл, Татнефть, МТС, Сбербанк

### Пример
```
Лукойл:
D1 = 1 100 руб (прогноз дивидендов на 2026)
r = 18% (требуемая доходность)
g = 5% (средний рост дивидендов за 5 лет)

P = 1 100 / (0.18 - 0.05) = 8 462 руб

Текущая цена: 7 200 руб → НЕДООЦЕНЕНА (апсайд 17.5%)
```

### Двухфазная модель (H-модель)
```
P = D0 × (1+gL) / (r - gL) + D0 × H × (gS - gL) / (r - gL)

gS = краткосрочный темп роста (5–10 лет)
gL = долгосрочный темп роста (бессрочно)
H = половина периода высокого роста (лет)
```

## Residual Income (остаточный доход)

### Формула
```
Value = BV₀ + Σ (NI_t - BV_t-1 × CoE) / (1 + r)^t

BV₀ = текущая балансовая стоимость
NI_t = чистая прибыль в году t
CoE = стоимость собственного капитала (из CAPM)
r = ставка дисконтирования
```

### Логика
- Если компания зарабатывает ROE > CoE → создаёт стоимость
- Если ROE < CoE → уничтожает стоимость
- Остаточный доход = прибыль сверх того, что требует капитал

### Когда применять
- Компании без дивидендов
- Компании с отрицательным FCF (инвестиционная фаза)
- Банки (где FCF сложно считать)
- Подходит для: Яндекс, OZON, банки

### Пример для банка
```
Сбербанк:
BV₀ = 310 руб/акцию
Прогноз NI = 75 руб/акцию
CoE = 18%
Остаточный доход = 75 - 310 × 0.18 = 75 - 55.8 = 19.2 руб

Компания создаёт стоимость (ROE 24% > CoE 18%)
```

## SOTP — Sum-of-the-Parts (поэтажная оценка)

### Метод
Для холдингов и конгломератов — оценить каждый бизнес-сегмент отдельно.

### Пример: АФК Система
```
Актив                  Доля    Оценка доли (млрд руб)
МТС                    50.0%   380
Segezha Group          62.2%   15
Медси                  95.5%   40
Степь                  91.4%   25
Биннофарм              75.0%   35
Cosmos Hotel Group     100%    20
Элемент                50.0%   30
Прочие активы          —       50
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Сумма активов                  595
Минус: чистый долг корп. центра -250
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
NAV до дисконта                345
Конгломератный дисконт (15%)   -52
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Справедливая стоимость         293 млрд руб
На акцию (9.65 млрд шт)       ~30 руб
```

### Конгломератный дисконт
- Типичный: 10–20% к суммарной стоимости частей
- В РФ: 13–15% (из-за непрозрачности и сложности управления)
- Может сужаться при IPO дочек или продаже активов

## NAV — чистые активы на акцию

### Формула
```
NAV = (Активы - Обязательства) / Количество акций

Или: NAV = Собственный капитал / Количество акций
```

### Когда P/NAV имеет смысл
| Тип компании | Типичный P/NAV | Комментарий |
|-------------|---------------|-------------|
| Банки | 0.5–1.5 | Основной показатель (= P/BV) |
| Девелоперы | 0.3–1.0 | NAV = стоимость земельного банка |
| Инвестиционные | 0.7–1.2 | NAV = портфель инвестиций |
| Страховые | 0.8–1.3 | NAV = резервы + инвестпортфель |

### Модифицированный NAV (mNAV) для девелоперов
```
mNAV = Рыночная стоимость земельного банка
     + Рыночная стоимость строящихся проектов
     + Денежные средства
     - Долг
     - Дисконт на ликвидность (10–20%)

Пример: ПИК, Самолёт, Эталон
```

## Сравнение методов: когда какой применять

| Метод | Лучше всего для | Не подходит для |
|-------|----------------|-----------------|
| DCF + WACC | Стабильные FCF, предсказуемый бизнес | Стартапы, циклические |
| CAPM | Оценка требуемой доходности | Нерыночные компании |
| Gordon | Дивидендные аристократы | Компании без дивидендов |
| Residual Income | Банки, инвестфаза | Убыточные компании |
| SOTP | Холдинги, конгломераты | Монобизнес |
| NAV | Банки, девелоперы, фонды | IT, сервисные компании |

### Рекомендация для бота
Использовать 2–3 метода для кросс-проверки. Если оценки расходятся > 30% — перепроверить допущения.
