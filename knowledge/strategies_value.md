# Стоимостное инвестирование (Value Investing)

> Покупай доллар за 50 центов. Ищи великие компании по справедливой цене или хорошие компании по отличной цене.

---

## Warren Buffett — экономические рвы и качество

### 5 типов экономического рва (Moat)

| Тип рва | Описание | Пример |
|----------|----------|--------|
| **Бренд** | Потребитель платит премию за имя | Coca-Cola, Apple |
| **Переключение** | Высокая стоимость смены поставщика | Oracle, 1C |
| **Сетевой эффект** | Ценность растёт с числом пользователей | Visa, Facebook |
| **Стоимостное преимущество** | Производишь дешевле всех | Walmart, Costco |
| **Нематериальные активы** | Патенты, лицензии, регуляторные барьеры | Фармацевтика, банки |

### Owner Earnings (истинная прибыль владельца)

```
Owner Earnings = Net Income + D&A - Maintenance CapEx

где:
  Net Income     — чистая прибыль
  D&A            — амортизация (возврат неденежных расходов)
  Maintenance CapEx — капзатраты на поддержание (НЕ на рост)

Приблизительно: Maintenance CapEx ≈ 50-70% от Total CapEx
```

### Критерии Баффетта

| Параметр | Порог |
|----------|-------|
| ROE | > 15% стабильно 5+ лет |
| Debt/Equity | < 0.5 (лучше < 0.3) |
| Profit Margin | > 20% (чем выше, тем сильнее moat) |
| Margin of Safety | 25-50% от intrinsic value |
| Горизонт | "Любимый срок удержания — навсегда" |
| Free Cash Flow | Стабильно растёт год к году |
| Менеджмент | Честный, компетентный, акционер сам |

### Принципы Баффетта

1. **Круг компетенции** — инвестируй только в то, что понимаешь
2. **Margin of Safety** — покупай значительно ниже внутренней стоимости
3. **Долгосрочность** — не продавай из-за краткосрочных колебаний
4. **Концентрация** — лучше 10 отличных идей, чем 50 средних
5. **Жадность и страх** — будь жадным, когда другие боятся

## Benjamin Graham — отец стоимостного инвестирования

### Graham Number (максимальная справедливая цена)

```
Graham Number = √(22.5 × EPS × BVPS)

где:
  EPS  — прибыль на акцию (за последние 12 мес)
  BVPS — балансовая стоимость на акцию
  22.5 = 15 (макс P/E) × 1.5 (макс P/B)

Покупать: текущая цена < Graham Number
```

### Intrinsic Value по Грэму

```
Intrinsic Value = EPS × (8.5 + 2g) × 4.4 / Y

где:
  EPS  — прибыль на акцию (TTM)
  8.5  — базовый P/E для компании с нулевым ростом
  g    — ожидаемый рост EPS на 7-10 лет (в %)
  4.4  — средняя доходность AAA-облигаций в эпоху Грэма (%)
  Y    — текущая доходность AAA-облигаций (%)

Margin of Safety: покупать при цене < 0.67 × Intrinsic Value
```

### Net-Net (NCAV) — стратегия сигарного окурка

```
NCAV = Current Assets - Total Liabilities

(НЕ вычитаем долгосрочные активы — только оборотные минус ВСЕ обязательства)

Покупать: цена акции < 0.67 × NCAV на акцию
Продавать: цена выросла на 50% или через 2-3 года
```

### Критерии Грэма (The Defensive Investor)

| Параметр | Порог |
|----------|-------|
| P/E | < 15 |
| P/B | < 1.5 |
| P/E × P/B | < 22.5 |
| Current Ratio | > 2.0 |
| Long-term Debt | < Net Current Assets |
| Дивиденды | Платит 20+ лет подряд |
| Рост EPS | > 33% за 10 лет (≈ 3% CAGR) |
| Размер | Выручка > $500M (крупные компании) |

## Charlie Munger — качество по справедливой цене

### Философия

- **"Лучше купить великую компанию по справедливой цене, чем посредственную по отличной"**
- Перевернул подход Грэма: от "дёшево" к "качественно"

### Ментальные модели (ключевые для инвестирования)

| Модель | Применение |
|--------|-----------|
| **Инверсия** | "Как потерять деньги?" → избегай этого |
| **Второго порядка** | Каковы последствия последствий? |
| **Круг компетенции** | Знай границы своего знания |
| **Lollapalooza** | Несколько факторов совпали — мощный эффект |
| **Opportunity cost** | Сравнивай с лучшей альтернативой, не с "ничего" |

### Концентрированный портфель

```
Правило Мангера:
  - 3-5 позиций = основа портфеля (60-80%)
  - Максимум 10 позиций всего
  - Одна великая идея в год — уже хорошо
```

### Fat Pitch (идеальный момент)

- Не торопись, жди идеальную подачу
- Критерии fat pitch:
  1. Бизнес, который полностью понимаешь
  2. Устойчивое конкурентное преимущество
  3. Менеджмент высокого качества
  4. Цена значительно ниже внутренней стоимости
  5. Высокая убеждённость (conviction > 80%)

## Чеклист стоимостного инвестора

### Перед покупкой

- [ ] Понимаю ли я бизнес? (могу объяснить 10-летнему)
- [ ] Есть ли экономический ров? Какой тип?
- [ ] ROE > 15% стабильно 5+ лет?
- [ ] Долг/Капитал < 0.5?
- [ ] Free Cash Flow стабильно положительный?
- [ ] P/E < 15 или P/E × P/B < 22.5?
- [ ] Margin of Safety > 25%?
- [ ] Менеджмент — честный и компетентный?
- [ ] Готов ли я держать 5+ лет?
- [ ] Это в моём круге компетенции?

### Перед продажей

- [ ] Бизнес фундаментально ухудшился? (НЕ просто цена упала)
- [ ] Ров разрушен? (новая технология, регуляция)
- [ ] Менеджмент стал нечестным?
- [ ] Нашёл значительно лучшую альтернативу?
- [ ] Оценка стала абсурдно завышена? (P/E > 40 для стабильного бизнеса)

### Красные флаги (НЕ покупать)

- Компания убыточна 2+ года подряд
- Долг растёт быстрее выручки
- Менеджмент продаёт свои акции массово
- Частая смена аудиторов
- Выручка растёт, а FCF падает (агрессивный учёт)
- Слишком сложная структура бизнеса
