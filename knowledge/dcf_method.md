# Метод дисконтированных денежных потоков (DCF)

> Урок №9 — Метод дисконтированных денежных потоков DCF

## Основные принципы

1. Базируется на **прогнозе денежных потоков** компании в будущем
2. Базируется на принципе: **"деньги сегодня дороже, чем деньги завтра"**
3. Базируется на принципе **сложного процента**

### Пример сложного процента

- 1000 руб. под 10% = 1100 руб.
- 1100 руб. под 10% = 1210 руб.

---

## Ключевые формулы

### Будущая стоимость (Future Value)

```
FV(n) = PV * (1 + r)^n
```

Где:
- **FV(n)** — будущая стоимость через n периодов
- **PV** — стоимость сейчас (текущая)
- **r** — процентная ставка
- **n** — количество периодов

**Пример:** 1000 * (1 + 0.1)^34 = 25 547.67 руб.

### Дисконтирование (Present Value)

```
PV = FV(n) / (1 + r)^n
```

**Дисконтирование** — приведение будущих потоков денег к текущему моменту. Оценка будущих денег в настоящем.

**Пример:** 25 547.67 / (1 + 0.1)^34 = 1000 руб.

---

## Ключевые определения

### Ставка дисконтирования

Минимальная доходность в %, которую хочет получить инвестор. **Выше риск — выше ставка дисконтирования.**

### Терминальная стоимость (Terminal Value, TV)

Стоимость компании **за пределами прогнозного периода**.

```
TV = FV(n) * (1 + g) / (r - g)
```

Где:
- **FV(n)** — последний прогнозный денежный поток
- **r** — ставка дисконтирования
- **g** — темп роста компании "в бесконечность"

---

## Задача-пример

Выберите вариант А или Б:
- **А** — 1000 руб. сейчас и положить на накопительный счет на 14 дней с ежедневным начислением %, ставка = 5% в день
- **Б** — 2000 руб. через 14 дней

**Ответ:** В варианте А: 1000 * (1.05)^14 = 1979.93 руб. В варианте Б получите больше (2000 руб.)

---

## Практическое применение DCF

DCF — это дисконтированные к настоящему моменту денежные потоки компании в будущем. Сумма того, сколько компания принесет акционерам в деньгах в будущем, приведенная к настоящему моменту = **справедливая стоимость компании**.

### Алгоритм

1. Прогнозируем свободный денежный поток для акционеров (FCF/FCFE) на **5+ лет** вперед
2. Считаем **терминальную стоимость** (TV)
3. Дисконтируем сумму (FCFE за 5 периодов + TV) к текущему моменту
4. Получаем **"справедливую стоимость"** компании
5. Делим стоимость на количество акций = **справедливая стоимость акции**
