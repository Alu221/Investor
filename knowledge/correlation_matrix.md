## Корреляции между конкретными парами акций

### Высокая корреляция (> 0.7) — ИЗБЕГАТЬ в одном портфеле
```
Лукойл — Роснефть:        0.85  (оба нефтянка, одни драйверы)
Лукойл — Татнефть:        0.80
Северсталь — НЛМК:        0.82  (оба чёрная металлургия)
Северсталь — ММК:          0.78
Сбер — Совкомбанк:        0.75  (оба банки)
X5 — Магнит:              0.72  (оба ритейл)
```

Сбер — Лукойл:            0.60  (разные секторы, но оба цикличны)
Лукойл — Северсталь:      0.55  (сырьё, но разные рынки)
Сбер — Яндекс:            0.45  (банк vs IT)
МТС — Магнит:             0.45  (внутренний рынок)
Роснефть — Фосагро:       0.50  (оба экспортёры)

## Практические правила диверсификации

### Минимум 5 секторов в портфеле
```
Идеальный портфель из 10 акций (корреляция < 0.5 между парами):
1. Нефтегаз (1 компания): Лукойл
2. Банки (1): Сбер
3. Золото (1): Полюс
4. IT (1): Яндекс
5. Ритейл (1): X5
6. Удобрения (1): Фосагро
7. Чёрная мет. (1): Северсталь
8. Телеком (1): Ростелеком
9. Электроэнергетика (1): Интер РАО
10. Девелопмент (1): ДОМ.РФ

Средняя корреляция пар: ~0.30
Ожидаемое снижение волатильности vs 1 акция: -40%
```

### Анти-паттерны (НЕ делать)
```
❌ 3 нефтяника в портфеле (Лукойл + Роснефть + Татнефть) — корреляция 0.80+
❌ 2 металлурга + 1 нефтяник — корреляция с сырьём 0.60+
❌ Весь портфель из экспортёров — одинаковый валютный риск
❌ Только госкомпании — один регуляторный риск
```
