# Поведенческие финансы

## Когнитивные искажения инвестора

### 1. FOMO — страх упущенной выгоды (Fear of Missing Out)

**Что происходит**: Акция растёт 3 дня подряд, все в чатах обсуждают, и вы покупаете на пике.

**Как проявляется**:
```
- «Все уже заработали, а я нет!»
- Покупка после роста 30–50% без анализа
- Вход в хайповые идеи (мем-акции, IPO)
- Увеличение позиции после роста вместо фиксации
```

**Защита**:
```
✅ Составить план позиции ЗАРАНЕЕ (цена входа, размер, цель)
✅ Правило 24 часов: после импульса «хочу купить» — подождать сутки
✅ Если акция выросла > 20% — НЕ покупать, ждать откат
✅ Вести журнал сделок: записывать причину покупки ДО покупки
✅ Помнить: лучше упустить 10 возможностей, чем влететь в 1 ловушку
```

---

### 2. Loss Aversion — неприятие потерь

**Что происходит**: Потеря 100 000 руб ощущается в 2.5 раза сильнее, чем радость от прибыли 100 000 руб (Канеман и Тверски).

**Как проявляется**:
```
- Держите убыточную позицию месяцами («отрастёт»)
- Продаёте прибыльную слишком рано («зафиксирую пока плюс»)
- Отказываетесь от хороших сделок из-за страха потери
- Усредняетесь в падающую акцию без анализа
```

**Защита**:
```
✅ Stop-loss для спекуляций (выставлять сразу при покупке)
✅ Правило: если тезис сломался — продать, независимо от цены
✅ Думать в процентах, не в рублях (убыток 5% — это просто 5%)
✅ Перед усреднением: «Купил бы я эту акцию с нуля по текущей цене?»
✅ Раз в месяц: пересмотреть все убыточные позиции критически
```

---

### 3. Anchoring — якорение

**Что происходит**: Привязка к первой увиденной цене или к цене покупки.

**Как проявляется**:
```
- «Сбер стоил 300, сейчас 270 — дёшево!» (а может, 270 — справедливо)
- «Я купил за 500, не продам ниже 500» (цена покупки не имеет значения)
- «Аналитик сказал целевая 400 — значит вырастет» (один прогноз = якорь)
- Привязка к историческим максимумам/минимумам
```

**Защита**:
```
✅ Пересчитывать fair value каждый квартал (новые данные = новая оценка)
✅ Забыть цену покупки при принятии решений
✅ Спрашивать: «Если бы у меня не было этой акции, купил бы я её сейчас?»
✅ Использовать 2–3 метода оценки, не один прогноз
✅ Игнорировать «целевые цены» аналитиков как единственный ориентир
```

---

### 4. Overconfidence — самоуверенность

**Что происходит**: После серии удачных сделок кажется, что вы гений рынка.

**Как проявляется**:
```
- Увеличение размера позиций (all-in в одну идею)
- Отказ от диверсификации («я точно знаю, что вырастет»)
- Игнорирование рисков и контраргументов
- Частые сделки (overtrading) — «я чувствую рынок»
- Использование плеча (маржинальная торговля)
```

**Защита**:
```
✅ Минимум 5–7 позиций в портфеле ВСЕГДА
✅ Максимум 10% на одну идею (даже если на 100% уверен)
✅ Вести статистику: считать Win Rate и средний R/R
✅ Перед каждой сделкой: записать 3 причины, почему можешь ошибаться
✅ Помнить: 80% активных трейдеров проигрывают индексу
```

---

### 5. Herd Behavior — стадное поведение

**Что происходит**: Делать то, что делают все, потому что «все не могут ошибаться».

**Как проявляется**:
```
- Покупка популярных акций из чатов/каналов
- Продажа в панике, когда все продают
- Следование за толпой в IPO
- «Если все покупают — значит это хорошо»
```

**Защита**:
```
✅ Contrarian мышление: «Если все уверены — я должен сомневаться»
✅ Цитата Баффетта: «Бойтесь, когда другие жадны. Жадничайте, когда другие боятся»
✅ Свой анализ ПЕРЕД чтением чужих мнений
✅ Избегать инвест-чатов в моменты эйфории и паники
✅ Покупать, когда рынок в страхе (VIX > 30, индексы на минимумах)
```

---

### 6. Recency Bias — переоценка недавних событий

**Что происходит**: Последние события кажутся важнее исторических данных.

**Как проявляется**:
```
- «Рынок падал 3 месяца — будет падать дальше» (может развернуться)
- «Компания 2 квартала растёт — будет расти вечно» (цикличность)
- Экстраполяция последнего тренда в бесконечность
- Переоценка одного плохого отчёта при хорошей 5-летней истории
```

**Защита**:
```
✅ Смотреть данные за 5–10 лет, не за последний квартал
✅ Помнить о цикличности (рынок падает → рынок растёт → рынок падает)
✅ Средние значения за 3–5 лет > последнее значение
✅ Вопрос: «Изменился ли бизнес ФУНДАМЕНТАЛЬНО или это временно?»
✅ Составить список: что было 1, 3, 5 лет назад (контекст)
```

---

### 7. Disposition Effect — эффект диспозиции

**Что происходит**: Продажа прибыльных слишком рано + удержание убыточных слишком долго.

**Как проявляется**:
```
Акция +15%: «Надо зафиксировать, пока не упала!» → продаёт
Акция -30%: «Подожду, отрастёт» → держит

Результат: портфель из убыточных позиций
```

**Защита**:
```
✅ Trailing stop: подтягивать стоп за растущей ценой
✅ Не продавать прибыльное без причины (пусть «бежит»)
✅ Регулярно чистить убыточные: «Купил бы с нуля?» — если нет, продать
✅ Правило: ограничивай убытки, не прибыль
```

---

### 8. Confirmation Bias — подтверждение своей правоты

**Что происходит**: Ищете только информацию, подтверждающую ваше решение.

**Как проявляется**:
```
- Купили Газпром → читаете только бычьи прогнозы
- Игнорируете негативные новости по своим позициям
- Спорите с теми, кто не согласен
```

**Защита**:
```
✅ Намеренно искать аргументы ПРОТИВ своей позиции
✅ Перед покупкой: прочитать 3 медвежьих аргумента
✅ Вопрос: «Что должно случиться, чтобы я признал ошибку?»
✅ Подписаться на аналитиков с РАЗНЫМИ мнениями
```

## Правила эмоциональной дисциплины

### 10 заповедей инвестора

```
1. ПЛАН → Каждая позиция имеет письменный тезис (почему купил, цель, стоп)
2. РАЗМЕР → Ни одна позиция не превышает 10% портфеля
3. СТОП → Stop-loss для спекуляций, пересмотр тезиса для инвестиций
4. ТЕРПЕНИЕ → Не торговать от скуки; лучшая сделка — та, которую не сделал
5. ДНЕВНИК → Вести журнал сделок (дата, тикер, причина, результат)
6. ПАУЗА → После крупного убытка — 1 неделя без сделок
7. ЛИМИТ → Не больше 2–3 сделок в неделю (для инвестора, не трейдера)
8. КОНТРАРГУМЕНТ → Перед покупкой записать 3 причины НЕ покупать
9. СКРОМНОСТЬ → Признавать ошибки быстро, фиксировать и идти дальше
10. АВТОМАТИЗАЦИЯ → Всё что можно автоматизировать (стопы, ребалансировка) — автоматизировать
```

### Чек-лист перед каждой сделкой
```
□ Это моё собственное решение (не подсказка из чата)?
□ Я могу объяснить тезис в 2 предложениях?
□ R/R > 1:3?
□ Размер позиции адекватный (< 10%)?
□ Я готов к убытку -30%?
□ Я не в состоянии эйфории или паники?
□ Я прочитал контраргументы?
```

### Сигналы, что пора остановиться
```
3 убыточные сделки подряд → перерыв 1 неделя
Проверяете котировки > 10 раз в день → отключить уведомления
Не можете спать из-за позиции → позиция слишком большая
Торгуете от злости/мести → немедленно остановиться
Используете плечо «чтобы отбить убытки» → КРАЙНЕ ОПАСНО
```

## Second-Level Thinking (Говард Маркс)

### Разница между первым и вторым уровнем мышления
```
Первый уровень: «Компания хорошая → покупаю»
Второй уровень: «Компания хорошая, но все это знают, цена завышена → не покупаю»

Первый уровень: «Рынок падает → продаю»
Второй уровень: «Рынок падает, все в панике, хорошие акции дёшевы → покупаю»

Первый уровень: «Дивиденды 15% → отличная акция»
Второй уровень: «Дивиденды 15%, но payout > 100% и долг растёт → ловушка»
```

### Вопросы для второго уровня
```
1. Какой консенсус рынка по этой акции?
2. Чем моё мнение отличается от консенсуса?
3. Кто продавец и почему он продаёт?
4. Что может пойти не так?
5. Какая вероятность, что я прав, а рынок нет?
```
