## Joel Greenblatt — Magic Formula

### Формула

```
Шаг 1: Рассчитать ROIC (Return on Invested Capital)

  ROIC = EBIT / Invested Capital

  где:
    EBIT = Operating Income (прибыль до процентов и налогов)
    Invested Capital = Net Working Capital + Net Fixed Assets
                     = (Current Assets - Current Liabilities) + (PP&E net)

  Порог: ROIC > 15% (чем выше, тем лучше)

Шаг 2: Рассчитать Earnings Yield

  Earnings Yield = EBIT / Enterprise Value

  где:
    Enterprise Value = Market Cap + Total Debt - Cash

  Порог: EY > 10% (чем выше, тем дешевле)

Шаг 3: Ранжирование

  Ранг_ROIC   — от 1 (лучший) до N
  Ранг_EY     — от 1 (лучший) до N
  Итоговый ранг = Ранг_ROIC + Ранг_EY

  Чем МЕНЬШЕ итоговый ранг — тем лучше акция
```

### Правила применения

```
1. Отфильтровать: Market Cap > $50M (или >10 млрд руб для РФ)
2. Исключить: финансовый сектор, utilities (ROIC нерелевантен)
3. Ранжировать все оставшиеся по Magic Formula
4. Купить топ 20-30 акций равными долями
5. Держать 1 год
6. Через год — продать, пересчитать, купить новый топ-30
7. Повторять минимум 3-5 лет (1-2 года могут отставать от рынка)

Налоговая оптимизация:
  - Акции в прибыли: продавать после 13 мес (льгота в РФ после 3 лет)
  - Акции в убытке: продавать до 12 мес (зафиксировать убыток)
```

### Результаты Magic Formula (исторические)

```
Бэктест Greenblatt (1988-2004, USA):
  Magic Formula: ~30.8% годовых
  S&P 500:       ~12.4% годовых

ВАЖНО: после публикации книги (2005) доходность снизилась,
но стратегия продолжает обыгрывать рынок на 3-5% в год
```

## Factor Investing — 5 факторов Fama-French

### Факторы и их премии

| Фактор | Определение | Историческая премия | Метрика |
|--------|------------|-------------------|---------|
| **Value** | Дешёвые обгоняют дорогие | +3-5% годовых | P/B, P/E, EV/EBIT |
| **Size** | Малые обгоняют крупные | +2-3% годовых | Market Cap |
| **Momentum** | Растущие продолжают расти | +4-6% годовых | 12-мес доходность |
| **Quality** | Прибыльные обгоняют убыточные | +3-4% годовых | ROE, стабильность EPS |
| **Low Volatility** | Низковолатильные обгоняют высоко | +2-3% годовых | σ за 12 мес |

### Модель Fama-French (5 факторов)

```
R_i - R_f = α + β₁(R_m - R_f) + β₂(SMB) + β₃(HML) + β₄(RMW) + β₅(CMA) + ε

где:
  R_i - R_f  — избыточная доходность акции над безрисковой ставкой
  R_m - R_f  — рыночная премия (Market)
  SMB        — Small Minus Big (фактор размера)
  HML        — High Minus Low (фактор стоимости, по P/B)
  RMW        — Robust Minus Weak (фактор прибыльности)
  CMA        — Conservative Minus Aggressive (фактор инвестиций)
  α          — альфа (то, что не объясняется факторами)
```

## Momentum — стратегия импульса

### Расчёт Momentum Score

```
Momentum Score = (Price_today - Price_12m_ago) / Price_12m_ago × 100%

Вариант с пропуском последнего месяца (избежать mean reversion):
  Momentum = (Price_1m_ago - Price_13m_ago) / Price_13m_ago × 100%

Интерпретация:
  > +30%  — сильный восходящий тренд (покупать)
  > +20%  — умеренный тренд (покупать)
  +5..+20% — слабый тренд (нейтрально)
  -5..+5%  — боковик (не входить)
  < -20%  — сильный нисходящий тренд (избегать или шортить)
```

### Правила Momentum-стратегии

```
1. Рассчитать Momentum Score для всех акций вселенной
2. Ранжировать от лучшего к худшему
3. Купить топ 10-20% (децильный портфель)
4. Ребалансировка: ежемесячно или ежеквартально
5. Если акция выпала из топ-30% — продать
6. Стоп-лосс: -15% от цены покупки (momentum может резко развернуться)

Фильтр качества (избежать мусорный momentum):
  - Market Cap > $100M
  - Average Volume > $1M/день
  - Не пенни-стоки (цена > $5)
```

### Dual Momentum (Gary Antonacci)

```
Абсолютный Momentum: актив обгоняет безрисковую ставку?
  Если доходность за 12 мес > ставка T-Bills → да
  Если нет → в кэш/облигации

Относительный Momentum: какой актив лучше?
  Сравнить: акции US vs акции International
  Купить тот, что показал лучший 12-мес результат

Комбинация:
  1. Выбрать лучший актив (относительный momentum)
  2. Если он > безрисковой ставки → инвестировать в него
  3. Если нет → 100% в облигации
  4. Пересматривать ежемесячно
```

## Mean Reversion — возврат к среднему

### Z-Score (основной индикатор)

```
Z-score = (Price - SMA_200) / StdDev_200

где:
  Price      — текущая цена
  SMA_200    — 200-дневная скользящая средняя
  StdDev_200 — стандартное отклонение цены за 200 дней

Сигналы:
  Z < -2.0  — СИЛЬНЫЙ BUY (экстремально ниже среднего)
  Z < -1.5  — BUY (значительно ниже среднего)
  Z = -1..+1 — нейтральная зона (без сигнала)
  Z > +1.5  — SELL (значительно выше среднего)
  Z > +2.0  — СИЛЬНЫЙ SELL (экстремально выше среднего)
```

### RSI как индикатор Mean Reversion

```
RSI = 100 - [100 / (1 + RS)]
RS = Average Gain (14 дней) / Average Loss (14 дней)

Сигналы:
  RSI < 30  — перепродано (buy)
  RSI < 20  — сильно перепродано (strong buy)
  RSI > 70  — перекуплено (sell)
  RSI > 80  — сильно перекуплено (strong sell)

Комбинация с Z-score:
  Z < -2 И RSI < 30 → двойное подтверждение → высоковероятный buy
```

### Bollinger Bands как инструмент Mean Reversion

```
Upper Band = SMA_20 + 2 × StdDev_20
Lower Band = SMA_20 - 2 × StdDev_20
%B = (Price - Lower Band) / (Upper Band - Lower Band)

Сигналы:
  %B < 0    — цена ниже нижней полосы → buy (возврат внутрь)
  %B > 1    — цена выше верхней полосы → sell
  %B = 0.5  — точно на средней линии → нейтрально
```

## Комбинирование факторов

### Multi-Factor Score

```
Итоговый Score = w₁×Value_Rank + w₂×Quality_Rank + w₃×Momentum_Rank

Варианты весов:
  Равные:     w₁ = w₂ = w₃ = 0.33
  Value-tilt: w₁ = 0.5, w₂ = 0.3, w₃ = 0.2
  Momentum:   w₁ = 0.2, w₂ = 0.3, w₃ = 0.5

Конкретные метрики для рангов:
  Value_Rank:    EV/EBIT (чем ниже, тем лучше)
  Quality_Rank:  ROIC (чем выше, тем лучше)
  Momentum_Rank: 12-мес доходность (чем выше, тем лучше)
```

### Какие факторы комбинировать

```
Хорошие комбинации (низкая корреляция между факторами):
  ✅ Value + Momentum      — "дешёвое, что растёт"
  ✅ Value + Quality       — "дешёвое и качественное" (Buffett по сути)
  ✅ Quality + Momentum    — "качественное, что растёт"
  ✅ Value + Quality + Momentum — тройная комбинация (лучший результат)

Плохие комбинации:
  ❌ Value + Size — оба склонны к одним и тем же акциям
  ❌ Momentum один — слишком высокий turnover и просадки
```

### Практическая реализация

```
Шаг 1: Определить вселенную (РФ: MOEX, ликвидные акции с оборотом > 10М руб/день)
Шаг 2: Рассчитать метрики по каждому фактору для всех акций
Шаг 3: Ранжировать (1 = лучший) по каждому фактору
Шаг 4: Рассчитать Multi-Factor Score
Шаг 5: Купить топ 15-20 акций равными долями
Шаг 6: Ребалансировать 1 раз в квартал
Шаг 7: Вести лог результатов, сравнивать с бенчмарком (IMOEX)
```

### Подводные камни количественных стратегий

- **Overfitting** — стратегия работает на истории, но не в реальности
- **Data mining** — нашёл паттерн случайно, приписал значение
- **Regime change** — стратегия работала 20 лет, но мир изменился
- **Transaction costs** — частая ребалансировка съедает прибыль
- **Capacity** — стратегия перестаёт работать когда слишком много денег
- **Survivorship bias** — тестируешь только на выживших компаниях
