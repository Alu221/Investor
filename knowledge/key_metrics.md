# Основные метрики бизнеса

> Урок №6 — Основные метрики бизнеса

## 1. Капитализация (Mcap)

```
Капитализация = кол-во акций * цена акции
```

*Кол-во акций см. либо на сайте компании, либо в примечаниях к "капиталу"*

## 2. Чистый долг (Net Debt)

```
Чистый долг = долг - кэш
```

## 3. Стоимость бизнеса (Enterprise Value, EV)

```
EV = капитализация + чистый долг
```

## 4. EBITDA

```
EBITDA = Операционная прибыль + Амортизация
```

*Амортизацию можно взять в отчете о движении денежных средств*

## 5. Маржинальность (Margin)

Рассчитывается на нескольких уровнях:

- **Валовая маржинальность** = Валовая прибыль / Выручка
- **Операционная маржинальность** = Операционная прибыль / Выручка
- **EBITDA маржинальность** = EBITDA / Выручка
- **Маржинальность до налога** = Прибыль до налога / Выручка
- **Чистая маржинальность** = Чистая прибыль / Выручка

## 6. Свободный денежный поток (FCF)

```
FCF = Операционный денежный поток (CFO) - Капитальные вложения (Capex)
```

*Есть много разных формул, но эта — самая простая. Некоторые компании FCF считают по-своему, например, Лукойл.*

## 7. Свободный денежный поток для акционеров (FCFE)

```
FCFE = FCF - уплаченные проценты - погашенный долг + выпущенный долг
```

FCFE отражает сумму, которая остается в распоряжении акционеров и может быть направлена на **дивиденды** или **обратный выкуп**.

## Нормы по секторам (российский рынок)

### ROE (Return on Equity)
| Сектор | Хорошо | Отлично | Плохо |
|--------|--------|---------|-------|
| Банки | > 15% | > 20% | < 10% |
| Нефтегаз | > 12% | > 18% | < 8% |
| Металлургия | > 15% | > 25% | < 10% |
| Ритейл | > 15% | > 25% | < 10% |
| IT/Интернет | > 10% | > 15% | < 5% |
| Связь | > 15% | > 20% | < 10% |

### EBITDA маржинальность
| Сектор | Норма | Высокая | Низкая |
|--------|-------|---------|--------|
| Нефтегаз | 25-35% | > 40% | < 20% |
| Металлургия (чёрная) | 20-30% | > 35% | < 15% |
| Золотодобыча | 50-65% | > 65% | < 40% |
| Удобрения | 40-50% | > 50% | < 30% |
| Ритейл | 7-12% | > 12% | < 5% |
| Банки | N/A | N/A | N/A |
| Связь | 35-45% | > 45% | < 30% |
| IT/Интернет | 15-25% | > 25% | < 10% |

### Owner Earnings (Buffett)
```
Owner Earnings = Чистая прибыль + Амортизация - Поддерживающий CapEx
```
Более точная метрика реальных денег для владельца, чем FCF.
