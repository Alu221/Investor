# Скоринговые модели оценки акций

## Piotroski F-Score (0–9 баллов)

Оценка финансового здоровья компании по 9 бинарным критериям (1 или 0).

### Прибыльность (4 балла)
1. **ROA > 0** — компания прибыльна (Чистая прибыль / Активы > 0)
2. **OCF > 0** — операционный денежный поток положительный
3. **ΔROA > 0** — ROA текущего года выше прошлого (прибыльность растёт)
4. **OCF > Net Income** — качество прибыли: денежный поток больше чистой прибыли (нет манипуляций)

### Леверидж и ликвидность (3 балла)
5. **ΔDebt/Assets < 0** — долговая нагрузка снижается
6. **ΔCurrent Ratio > 0** — текущая ликвидность растёт (оборотные активы / краткосрочные обязательства)
7. **Нет разводнения** — количество акций не увеличилось (нет допэмиссии)

### Операционная эффективность (2 балла)
8. **ΔGross Margin > 0** — валовая маржа растёт (улучшение ценообразования)
9. **ΔAsset Turnover > 0** — оборачиваемость активов растёт (Выручка / Активы)

### Интерпретация
| Баллы | Сигнал | Действие |
|-------|--------|----------|
| 8–9 | Сильная компания | Покупать / держать |
| 5–7 | Средняя | Анализировать детальнее |
| 3–4 | Слабая | Осторожность |
| 0–2 | Очень слабая | Избегать / шортить |

### Пример расчёта для российской компании
```
Лукойл 2025:
ROA > 0?          ДА (12.3%) → 1
OCF > 0?          ДА (850 млрд) → 1
ΔROA > 0?         ДА (11.8% → 12.3%) → 1
OCF > NI?         ДА (850 > 720) → 1
ΔDebt/Assets < 0? ДА (0.15 → 0.12) → 1
ΔCR > 0?          НЕТ (1.8 → 1.7) → 0
Нет разводнения?  ДА → 1
ΔGross Margin > 0? ДА → 1
ΔAsset Turn > 0?  ДА → 1
Итого: 8/9 — сильная компания
```

## Altman Z-Score (вероятность банкротства)

### Формула для публичных компаний
```
Z = 1.2×K1 + 1.4×K2 + 3.3×K3 + 0.6×K4 + 1.0×K5

K1 = Рабочий капитал / Активы
K2 = Нераспределённая прибыль / Активы
K3 = EBIT / Активы
K4 = Рыночная капитализация / Общий долг
K5 = Выручка / Активы
```

### Интерпретация
| Z-Score | Зона | Вероятность банкротства |
|---------|------|------------------------|
| Z > 3.0 | Безопасная (зелёная) | Низкая (< 5%) |
| 1.8 < Z < 3.0 | Серая зона | Умеренная (15–30%) |
| Z < 1.8 | Опасная (красная) | Высокая (> 50%) |

### Модификация для развивающихся рынков (Altman EM Z-Score)
```
Z'' = 3.25 + 6.56×K1 + 3.26×K2 + 6.72×K3 + 1.05×K4

Безопасная: Z'' > 2.6
Серая: 1.1 < Z'' < 2.6
Опасная: Z'' < 1.1
```

### Ограничения для РФ
- Не подходит для банков и финансовых компаний (другая структура баланса)
- Для банков использовать: достаточность капитала (Н1 > 12%), NPL ratio < 5%
- Рыночная капитализация РФ компаний может быть занижена из-за странового дисконта

## Beneish M-Score (выявление манипуляций)

Определяет вероятность манипуляции финансовой отчётностью.

### Формула
```
M = -4.84 + 0.920×DSRI + 0.528×GMI + 0.404×AQI + 0.892×SGI
    + 0.115×DEPI - 0.172×SGAI + 4.679×TATA - 0.327×LVGI
```

### 8 переменных
| Переменная | Что измеряет | Расчёт |
|-----------|-------------|--------|
| DSRI | Индекс дней дебиторки | (Дебиторка/Выручка)_t / (Дебиторка/Выручка)_t-1 |
| GMI | Индекс валовой маржи | Gross Margin_t-1 / Gross Margin_t |
| AQI | Индекс качества активов | (1 - CA + PPE)/Assets_t / (1 - CA + PPE)/Assets_t-1 |
| SGI | Индекс роста выручки | Выручка_t / Выручка_t-1 |
| DEPI | Индекс амортизации | Depreciation Rate_t-1 / Depreciation Rate_t |
| SGAI | Индекс SGA расходов | (SGA/Выручка)_t / (SGA/Выручка)_t-1 |
| TATA | Начисления к активам | (NI - OCF) / Total Assets |
| LVGI | Индекс левериджа | Leverage_t / Leverage_t-1 |

### Интерпретация
```
M < -2.22 → Честная отчётность (вероятность манипуляций < 3%)
M > -2.22 → Вероятные манипуляции (вероятность > 50%)
```

### Красные флаги для российских компаний
- Резкий рост DSRI (дебиторка растёт быстрее выручки) — могут рисовать выручку
- TATA > 0.05 — начисленная прибыль не подтверждена кэшем
- Частая смена аудиторов
- РСБУ и МСФО сильно расходятся

## DuPont анализ (декомпозиция ROE)

### 3-факторная модель
```
ROE = Net Margin × Asset Turnover × Equity Multiplier

ROE = (Net Income / Revenue) × (Revenue / Assets) × (Assets / Equity)
```

| Фактор | Что показывает | Как улучшить |
|--------|---------------|-------------|
| Net Margin | Прибыльность | Снизить расходы, поднять цены |
| Asset Turnover | Эффективность активов | Больше выручки с тех же активов |
| Equity Multiplier | Финансовый рычаг | Больше долга (но выше риск!) |

### 5-факторная модель (расширенная)
```
ROE = (EBT/EBIT) × (EBIT/Revenue) × (Revenue/Assets) × (Assets/Equity) × (NI/EBT)

ROE = Налоговое бремя × Операционная маржа × Оборачиваемость × Леверидж × Процентное бремя
```

### Практическое применение
```
Пример: Сбербанк vs Тинькофф

Сбербанк:  ROE = 24% = 28% × 0.03 × 10 × 9
           (высокая маржа, низкая оборачиваемость, высокий леверидж)

Тинькофф:  ROE = 30% = 35% × 0.05 × 7 × 8
           (ещё выше маржа, выше оборачиваемость, меньше леверидж)

Вывод: Тинькофф зарабатывает ROE эффективнее, с меньшим риском
```

## Комбинированный скоринг для бота

### Финальная оценка компании (0–100 баллов)
```
Итоговый скор =
  Piotroski (0–9) × 3        = макс 27 баллов
  + Z-Score бонус             = макс 15 баллов
  + M-Score бонус             = макс 10 баллов
  + DuPont качество ROE       = макс 18 баллов
  + Мультипликаторы           = макс 30 баллов

Z-Score бонус:
  Z > 3.0 → 15 баллов
  1.8–3.0 → 8 баллов
  < 1.8 → 0 баллов

M-Score бонус:
  M < -2.22 → 10 баллов
  -2.22 ≤ M < -1.78 → 5 баллов
  M > -1.78 → 0 баллов
```
