## Почему РФ торгуется с дисконтом 50–70%

### Структурные причины
```
1. Санкции: ограничения на инвестиции, заморозка резервов ЦБ
2. Отток нерезидентов: до 2022 владели ~50% free-float, сейчас ~0%
3. Закрытый рынок капитала: нет притока западных денег
4. Страновой риск: геополитическая премия 5–8% к WACC
5. Корпоративное управление: риск «указаний сверху» (Газпром, ВТБ)
6. Ограниченная ликвидность: дневной оборот MOEX ~50–80 млрд руб
```

## Контрарианский взгляд (Templeton)

### Когда EM-рынок интересен по CAPE
```
CAPE < 10  — экстремально дёшев → покупать (Россия, Турция)
CAPE 10–15 — дёшев → присматриваться (Бразилия, Китай)
CAPE 15–25 — справедливо → держать
CAPE > 25  — дорог → не наращивать (Индия)
```

### РФ: аргументы ЗА покупку на текущих уровнях
```
+ P/E ~5 при ROE ~18% — аномально дёшево
+ Дивидендная доходность ~10% (выше ставки по USD-облигациям)
+ Компании без долга (Лукойл, Сургут, металлурги)
+ Импортозамещение — долгосрочный драйвер для IT-сектора
+ Начало снижения ставки ЦБ → переток с депозитов в акции
```

### РФ: аргументы ПРОТИВ
```
- Санкции могут ужесточиться
- Нет механизма раскрытия стоимости (нерезиденты не придут)
- Корпоративное управление: решения в пользу государства, не миноритариев
- Делистинг-риски (принудительные выкупы по заниженным ценам)
- Налоговая нагрузка на компании может расти (windfall tax, НДПИ)
```

## Практический вывод для инвестора

```
Россия — рынок для тех, кто:
  ✅ Готов к высокой волатильности
  ✅ Понимает страновые риски
  ✅ Инвестирует на 3–5+ лет
  ✅ Фокусируется на дивидендах (cash return > рост цены)

Оптимальный подход:
  - Дивидендные фишки с низким долгом (Лукойл, Сбер, Татнефть)
  - Максимум 50–70% портфеля в РФ (остальное: золото, ОФЗ, валютные активы)
  - Ребалансировка раз в квартал
```
