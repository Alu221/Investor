# Сравнительный подход: мультипликаторы

> Урок №8 — Количественные подходы к оценке бизнеса. Мультипликаторы.

## Три подхода к оценке бизнеса

1. **Затратный подход** — сколько денег необходимо потратить, чтобы заново воссоздать такой актив
2. **Доходный подход (DCF)** — сколько денег компания нам принесет в будущем
3. **Сравнительный подход (мультипликаторы)** — сравнение компании с аналогами-конкурентами по отрасли

---

## Основные мультипликаторы

### P/E — Price to Earnings

```
P/E = Капитализация / Чистая прибыль
```
Сколько годовых прибылей стоит компания.

### P/S — Price to Sales

```
P/S = Капитализация / Выручка
```
Сколько годовых выручек стоит компания. Используется для оценки **компаний роста**, где еще нет прибыли.

### P/BV — Price to Book Value

```
P/BV = Капитализация / Капитал
```
Сколько капиталов стоит компания. Используется для **финансовых компаний** (страховые, банки).

### P/FCF — Price to Free Cash Flow

```
P/FCF = Капитализация / Свободный денежный поток
```
Сколько годовых денежных потоков стоит компания на бирже.

### EV/EBITDA — Enterprise Value to EBITDA

```
EV/EBITDA = Стоимость бизнеса / EBITDA
```
Сколько годовых EBITDA стоит компания.

### ND/EBITDA — Net Debt to EBITDA (Долговая нагрузка)

```
ND/EBITDA = Чистый долг / EBITDA
```
Главный мультипликатор для оценки **долговой нагрузки**. Если ND/EBITDA > 3x — у компании высокая долговая нагрузка (требуется >= 3 годовых EBITDA для покрытия текущего долга).

### ROE — Return on Equity

```
ROE = Чистая прибыль / Капитал
```
Показывает эффективность использования капитала. ROE = 10% означает, что на каждый используемый рубль собственных средств компания зарабатывает 10 копеек чистой прибыли. Чаще используется для **финансовых компаний**.

### ROA — Return on Assets

```
ROA = Чистая прибыль / Активы
```
Показывает эффективность использования активов.

---

## Подход LTM (Last Twelve Months)

В мультипликаторах используются **годовые** значения. Если последние данные за 2 или 3 квартал, нужно привести квартальные данные к годовым:

```
LTM = данные за последний квартал + данные за прошлый год - данные за соответствующий квартал прошлого года
```

**Пример:**
```
LTM EBITDA = EBITDA за 3кв 2023 + EBITDA за 2022 - EBITDA за 3кв 2022
```

## Суть сравнительного подхода

1. Посмотреть как рынок оценивает компанию **по отношению к аналогам-конкурентам** в отрасли
2. Сравнить текущую оценку со **средним историческим мультипликатором** по самой компании
3. Сравнить текущую оценку со **средним мультипликатором по отрасли**

## Ключевые комбинации для поиска недооценки

### Высокий ROE + низкий P/BV

Пример недооценки: **Сбербанк** 2020, 2022 (Сбер по 100 руб.)

### Высокие темпы роста + низкие мультипликаторы

Высокие темпы роста выручки/EBITDA/прибыли, но низкий P/E, EV/EBITDA или P/S — возможно компания недооценена, или есть другие причины.

**Примеры "других причин":**
- Mail.ru / VK — крайне низкий P/S для IT-компании
- ВТБ

## Важное правило

> Если у компании мультипликатор существенно меньше, чем у остальных и это продолжается уже длительное время (> 2-3 лет), то на это есть причины! Низкий мультипликатор — это **не значит**, что компания стоит дешево и именно ее нужно покупать. Это лишь дополнительный повод для анализа причин "почему так?"

## Красные флаги по мультипликаторам

| Мультипликатор | Значение | Что это может значить |
|----------------|----------|----------------------|
| P/E < 3 | Подозрительно дёшево | Рынок ждёт убытков, делистинг, или разовая прибыль |
| P/E < 5 | Очень дёшево | Проверить: циклическое дно? Value trap? Проблемы с менеджментом? |
| P/E > 25 | Дорого | Оправдано только если EPS рост > 20% (IT, e-commerce) |
| ND/EBITDA > 3x | Высокий долг | При растущей ставке ЦБ = КРИТИЧНО опасно |
| ND/EBITDA > 5x | Экстремальный долг | Риск банкротства, проверить Z-Score |
| P/BV < 0.5 | Глубокая скидка | Или отличная сделка, или рынок знает что-то плохое |
| ROE < 5% | Неэффективность | Деньги акционеров работают плохо |
| EV/EBITDA < 2 | Подозрительно | Проверить: временный всплеск EBITDA? Долг скрыт? |

## Отраслевые нормы мультипликаторов (MOEX)

| Сектор | P/E | EV/EBITDA | P/BV | Примечание |
|--------|-----|-----------|------|-----------|
| Нефтегаз | 4-8 | 2-5 | 0.5-1.5 | Циклический, зависит от нефти |
| Банки | 3-7 | N/A | 0.7-1.5 | Используй P/BV и ROE |
| Металлургия | 5-10 | 3-6 | 1-3 | Циклический сектор |
| Ритейл | 8-15 | 4-8 | 2-4 | Высокий P/E = рост + арендные обязательства |
| IT/Интернет | 15-30 | 8-15 | 2-5 | Дорого, но рост оправдывает |
| Связь | 5-12 | 3-5 | 0.5-1.5 | Стабильный, дивидендный |
| Удобрения | 8-15 | 4-7 | 3-6 | Высокие маржи, волатильность цен |
