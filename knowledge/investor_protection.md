# Защита инвестора на российском рынке

## Как хранятся акции

### НРД (Национальный расчётный депозитарий)
```
Инвестор → Брокер (торговый счёт) → Депозитарий брокера → НРД

НРД — центральный депозитарий России (100% дочка Мосбиржи).
Все акции MOEX хранятся в НРД на сегрегированных счетах.

Сегрегированное хранение:
  - Активы клиентов отделены от активов брокера
  - Брокер НЕ может использовать ваши акции для своих сделок
    (если вы не подписали договор на маржинальное кредитование)
```

## Брокер обанкротился — что с акциями?

### Акции = собственность клиента
```
Ваши акции ≠ активы брокера. При банкротстве:
1. Акции переводятся к другому брокеру (заявление в АСВ/конкурсному управляющему)
2. Срок: 1–3 месяца (практика: Энергокапитал, 2016)
3. Денежные средства на брокерском счёте = обычный кредитор (НЕ застрахованы)

Что защищено:         Что НЕ защищено:
✅ Акции на счёте      ❌ Кэш на брокерском счёте
✅ Облигации           ❌ Незавершённые сделки
✅ ОФЗ                 ❌ Маржинальные позиции (закроют принудительно)
```

### Отличие от банковских депозитов
| | Депозит | Брокерский счёт |
|---|---------|----------------|
| Страхование | АСВ до 1.4 млн руб | НЕТ страхования |
| При банкротстве | Выплата АСВ за 14 дней | Перевод акций 1–3 мес |
| Кэш | Застрахован | НЕ застрахован |
| Активы | Принадлежат банку | Принадлежат клиенту |

**В РФ нет аналога SIPC (США) — компенсационного фонда для инвесторов.**

## Заморозка активов 2022

### Что произошло
```
Февраль 2022: санкции → Euroclear/Clearstream заморозили активы
Пострадали:
  - Нерезиденты: акции РФ компаний заморожены на счетах НРД (~6 трлн руб)
  - Российские инвесторы: иностранные акции (через СПБ Биржу) заморожены

Российские акции на MOEX: НЕ заморожены для резидентов РФ
  → Торги возобновились 24 марта 2022 (через месяц паузы)
```

### Уроки
```
1. Страновой риск = реальный риск (не теоретический)
2. Иностранные акции через РФ-инфраструктуру = уязвимы
3. Диверсификация по юрисдикциям (не только по секторам)
```

## Как защитить себя

### Правила
```
✅ 2+ брокера (если один упадёт — второй работает)
✅ Не держать крупный кэш на брокерском счёте (выводить на банк)
✅ Сегрегированный счёт (по умолчанию, но проверить договор)
✅ Не подписывать договор займа ценных бумаг (брокер одалживает ваши акции)
✅ ИИС только у крупного брокера (Сбер, Т-банк, ВТБ, БКС)
✅ Хранить выписки из депозитария (подтверждение владения)
```

## Маржинальная торговля: почему опасно

### Механика
```
Плечо 1:2 → купил акций на 200 000, своих денег 100 000
Акция упала на 30%:
  Стоимость: 140 000
  Долг брокеру: 100 000
  Ваш капитал: 40 000 (убыток 60%!)

Плечо 1:5 → купил на 500 000, своих 100 000
Акция упала на 20%:
  Стоимость: 400 000
  Долг: 400 000
  Ваш капитал: 0 → MARGIN CALL → принудительная ликвидация
```

### Margin Call
```
Брокер принудительно закрывает позиции когда:
  Уровень маржи < минимально допустимого (обычно 15–25%)

Особенности:
  - Брокер закрывает по РЫНОЧНОЙ цене (в панике = по худшей)
  - Уведомление может не прийти (или прийти поздно)
  - Убыток может превысить депозит (вы должны брокеру!)
  - При гэпе открытия (gap down) стоп-лоссы не спасают
```

### Статистика
```
~80% маржинальных трейдеров теряют деньги
~15% выходят в ноль
~5% зарабатывают

Правило: маржинальная торговля — для профессионалов.
Долгосрочный инвестор НИКОГДА не использует плечо.
```
