# Источники данных для российского рынка

## Котировки и фундаментальные данные

### smart-lab.ru
```
Что даёт:
- Котировки акций и облигаций в реальном времени
- Фундаментальные данные: мультипликаторы, дивиденды, финансы
- Аналитика и блоги трейдеров/инвесторов
- Портфели пользователей (идеи)
- Дивидендный календарь
- Скринер акций по мультипликаторам

Ключевые страницы:
- /q/shares/ — все акции с мультипликаторами
- /dividends/ — дивидендный календарь
- /forum-pair/ — форум по конкретным акциям
- /q/TICKER/f/y/ — финансовые данные компании по годам

Плюсы: удобная агрегация, активное сообщество
Минусы: данные могут запаздывать, не всегда точные
```

### e-disclosure.ru (Центр раскрытия информации)
```
Что даёт:
- Официальное раскрытие информации эмитентами
- Отчёты МСФО и РСБУ (PDF)
- Существенные факты (дивиденды, buyback, сделки)
- Решения советов директоров
- Проспекты эмиссий

Применение:
- Первоисточник финансовой отчётности
- Проверка данных из других источников
- Корпоративные события до их появления в СМИ

Плюсы: официальный источник, полнота данных
Минусы: неудобный интерфейс, PDF-формат
```

### investing.com/ru
```
Что даёт:
- Макроэкономические данные (инфляция, ВВП, ставки)
- Экономический календарь (события, отчёты)
- Квартальные финансовые отчёты компаний
- Технический анализ (графики, индикаторы)
- Форекс, сырьё, криптовалюты
- Мультипликаторы зарубежных компаний

Ключевые разделы:
- /economic-calendar/ — экономический календарь
- /equities/ — котировки и анализ
- /rates-bonds/russia-government-bonds — ОФЗ

Плюсы: широкий охват, макроданные, калькуляторы
Минусы: реклама, часть данных с задержкой
```

## Дивиденды и доходность

### dohod.ru (ДОХОДЪ)
```
Что даёт:
- Дивидендный календарь (подробный)
- Индекс стабильности дивидендов (DSI)
- Рейтинги акций и облигаций
- Калькулятор облигаций
- Инвестиционные стратегии и ETF от УК ДОХОДЪ
- Анализ дивидендной политики компаний

Ключевые страницы:
- /ik/analytics/dividend/ — дивидендный календарь
- /ik/analytics/share/ — рейтинг акций
- /ik/analytics/bonds/ — рейтинг облигаций

Плюсы: лучший дивидендный ресурс в РФ, DSI-рейтинг
Минусы: не все компании покрыты
```

### snowball-income.com
```
Что даёт:
- Дивиденды на год вперёд (прогнозы)
- Дивидендная история компаний
- Сравнение дивидендных доходностей
- Портфельный трекер с учётом дивидендов
- Расчёт будущего дивидендного дохода

Плюсы: фокус на дивидендном инвестировании, удобные прогнозы
Минусы: часть функций платная
```

## Официальные биржевые данные

### moex.com (Московская биржа)
```
Что даёт:
- Официальные котировки акций, облигаций, валют
- Индексы (IMOEX, RTSI, отраслевые)
- Объёмы торгов
- Листинг и делистинг
- Корпоративные действия
- Маркетдата для профессионалов

Ключевые разделы:
- /ru/index/IMOEX/ — индекс МосБиржи
- /ru/listing/securities/ — все торгуемые инструменты
- /ru/marketdata/ — маркетдата

MOEX API:
  URL: https://iss.moex.com/
  Формат: JSON/XML
  Бесплатно, без авторизации
  Пример: https://iss.moex.com/iss/engines/stock/markets/shares/boards/TQBR/securities.json
```

### finam.ru
```
Что даёт:
- Исторические котировки (скачивание CSV)
- Технический анализ
- Новости и аналитика
- Скринер акций
- Экспорт данных для бэктестинга

Ключевое:
- /profile/moex-akcii/ — профили компаний
- Экспорт исторических данных — лучший бесплатный источник в РФ

Плюсы: исторические данные с 2000+ годов
Минусы: интерфейс устарел, навигация сложная
```

## Аналитика и новости

### rbc.ru/investments
```
Что даёт:
- Инвестиционные новости
- Аналитика от экспертов

- Дивидендные новости

Плюсы: оперативность, широкий охват
Минусы: не всегда глубокая аналитика
```

```
Что даёт:
- Финансовые отчёты компаний в удобном формате
- Сравнение компаний по мультипликаторам
- Графики финансовых показателей
- Исторические данные по выручке, прибыли, активам
- Скринер по фундаментальным показателям

Плюсы: удобная визуализация финансовых данных
Минусы: не все компании, обновление с задержкой
```

## API для автоматизации

### Tinkoff Invest API (gRPC)
```
Протокол: gRPC (+ REST gateway)
Документация: https://tinkoff.github.io/investAPI/
Авторизация: токен из приложения Т-Инвестиции

Возможности:
- Получение котировок (реальное время и исторические)
- Информация об инструментах (акции, облигации, фонды)
- Торговые операции (покупка/продажа)
- Портфель и операции
- Стаканы (order book)
- Свечи (candles) с таймфреймами от 1 мин до 1 месяца

Библиотеки:
- Python: tinkoff-investments (pip install tinkoff-investments)
- Go: invest-api-go-sdk
- C#: Tinkoff.InvestApi

Лимиты:
- Unary: 200 запросов/мин
- Stream: 300 подключений
```

### MOEX ISS API
```
Протокол: REST
URL: https://iss.moex.com/iss/
Формат: JSON, XML, CSV
Авторизация: не требуется (бесплатно)

Примеры запросов:
# Котировки всех акций на TQBR
GET /iss/engines/stock/markets/shares/boards/TQBR/securities.json

# Свечи по конкретной акции
GET /iss/engines/stock/markets/shares/boards/TQBR/securities/SBER/candles.json?from=2025-01-01&interval=24

# Информация об индексе IMOEX
GET /iss/engines/stock/markets/index/securities/IMOEX.json

# Дивиденды
GET /iss/securities/SBER/dividends.json

Лимиты: неофициально ~50 запросов/сек
Документация: https://iss.moex.com/iss/reference/
```

## Рекомендуемый workflow для анализа

```
1. Идея → smart-lab.ru (быстрый скрининг по мультипликаторам)
2. Финансы → conomy.ru или e-disclosure.ru (детальная отчётность)
3. Дивиденды → dohod.ru (история, DSI, прогноз)
4. Оценка → tbank.ru (консенсус аналитиков) + свой расчёт
5. Макро → investing.com (экономический календарь, ставки)
6. Риски → rbc.ru + smart-lab.ru (новости, обсуждения)
7. Решение → чек-лист из universal_checklist.md
```
