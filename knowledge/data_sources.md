# Источники данных для российского рынка

## Основные платформы

### smart-lab.ru
- Котировки акций/облигаций в реальном времени
- Фундаментальные данные (P/E, P/BV, ROE, дивиденды)
- Дивидендный календарь
- Портфели инвесторов с историей
- Блоги аналитиков
- Карта рынка (тепловая карта)

### e-disclosure.ru (Интерфакс)
- Официальное раскрытие информации эмитентами (закон)
- Квартальные и годовые отчёты МСФО/РСБУ
- Корпоративные события (дивиденды, реорганизация)
- Материалы ГОСА

### investing.com/ru
- Глобальные котировки (акции, облигации, валюты, сырьё)
- Экономический календарь (ставки ЦБ, инфляция, ВВП)
- Технический анализ и графики
- Финансовые отчёты компаний

### dohod.ru (УК ДОХОДЪ)
- Дивидендный календарь (самый полный для РФ)
- Историческая статистика дивидендов
- Рейтинги компаний по дивидендам
- Калькулятор доходности

### moex.com (Московская биржа)
- Официальные котировки и объёмы
- Индексы (IMOEX, RTS)
- Календарь корпоративных событий
- Список торгуемых инструментов

### tbank.ru (Т-Инвестиции)
- Удобный анализ компаний
- История дивидендов
- Калькулятор справедливой стоимости
- Новости и аналитика

### conomy.ru
- Финотчёты компаний в удобном табличном формате
- Мультипликаторы рассчитаны автоматически
- Сравнение компаний

### finam.ru
- Исторические данные (котировки за любой период)
- Экспорт данных
- Аналитика

### rbc.ru/investments
- Новости рынка
- Прогнозы аналитиков
- Котировки

### snowball-income.com
- Дивидендный календарь на год вперёд
- Прогнозы дивидендов
- Уведомления о датах отсечки

## API для автоматизации

### Tinkoff Invest API
- gRPC интерфейс
- SDK: Python, Node.js, Go
- Котировки, ордербук, история
- Бесплатно для клиентов

### MOEX ISS API
- REST API
- Котировки, объёмы, индексы
- Бесплатно, без регистрации
- Документация: iss.moex.com
