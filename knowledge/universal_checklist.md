# Универсальный чеклист перед покупкой акции

> Комбинация лучших практик: Buffett, Graham, Lynch, Greenblatt, Marks, Fisher, Dalio

## 1. Фундаментальный анализ

- [ ] P/E < 15-20 (или PEG < 1.0 для компаний роста)
- [ ] P/B < 2.0 (для банков P/BV ≈ 1.0 = справедливо)
- [ ] EV/EBITDA < 8 (для нефтегаза < 5, для IT допустимо 10-15)
- [ ] ROE > 15% (стабильно 3+ года)
- [ ] ND/EBITDA < 2x (< 3x допустимо, > 3x = опасно)
- [ ] FCF > 0 и растёт
- [ ] Дивиденды стабильны или растут
- [ ] Маржинальность EBITDA стабильна (не сжимается)

## 2. Качество бизнеса (Buffett / Munger / Fisher)

- [ ] Понимаю как компания зарабатывает деньги
- [ ] Есть экономический moat (бренд, масштаб, сети, патенты, switching costs)
- [ ] Менеджмент честный и компетентный
- [ ] Маржи стабильны 3+ года (не сжимаются)
- [ ] Рост органический (не за счёт долга)
- [ ] Дивидендная политика прозрачна

## 3. Скоринговые модели

- [ ] Piotroski F-Score ≥ 7 (из 9)
- [ ] Altman Z-Score > 3.0 (< 1.8 = банкротство)
- [ ] Beneish M-Score < -2.22 (> -2.22 = подозрение на манипуляции)
- [ ] Magic Formula: ROIC > 15% И Earnings Yield > 10%

## 4. Стратегическая проверка

- [ ] Graham Number: цена < √(22.5 × EPS × BVPS)
- [ ] Owner Earnings (Buffett) > 0 и растёт
- [ ] Lynch PEG < 1.0 (для компаний роста)
- [ ] Fisher: минимум 5 из 6 качественных критериев = "отлично"

## 5. Макро и цикл (Marks / Dalio / Templeton)

- [ ] Определена фаза экономического цикла
- [ ] Ставка ЦБ: какой тренд? (рост = осторожность, снижение = возможности)
- [ ] Нефть/рубль: как влияет на эту компанию?
- [ ] Санкционный риск минимален или учтён в цене
- [ ] Сектор не на пике цикла (second-level thinking)

## 6. Риск-управление

- [ ] Маржа безопасности 25-50% от fair value
- [ ] Это НЕ value trap (проверить: почему так дёшево?)
- [ ] Готов к просадке -30-50% (горизонт > 3 лет)
- [ ] Позиция ≤ 10% портфеля (max на одну идею)
- [ ] Есть план выхода (при каких условиях продаю?)
- [ ] Портфель диверсифицирован по 3+ секторам

## 7. Психологическая проверка (против себя)

- [ ] Это НЕ покупка из FOMO ("все купили, и я хочу")
- [ ] Это НЕ стадное поведение
- [ ] Применяю second-level thinking (не первую очевидную мысль)
- [ ] Специально искал аргументы ПРОТИВ покупки
- [ ] Готов холдить 3-5+ лет
- [ ] Инвестирую деньги, которые НЕ нужны в ближайшие 1-2 года

---

## Быстрая оценка (если нет времени на полный чеклист)

**Минимум 5 "да" из 7:**
1. P/E < 10?
2. ROE > 15%?
3. ND/EBITDA < 2x?
4. FCF > 0?
5. Дивиденды есть и стабильны?
6. Понимаю бизнес?
7. Есть маржа безопасности?

**Если 5+ "да"** → стоит рассмотреть детально.
**Если < 3 "да"** → пропустить.
