# Универсальный чеклист перед покупкой акции

> Комбинация лучших практик: Buffett, Graham, Lynch, Greenblatt, Marks, Fisher, Dalio, Piotroski

---

## 1. Фундаментальный анализ

### Оценка (дёшево ли?)
```
□ P/E < 15–20 (для стоимостных)
  Или PEG < 1.0 (для компаний роста: PEG = P/E ÷ темп роста прибыли)
□ P/B < 2.0
  Для банков: P/BV ~1.0 = справедливо, < 0.7 = недооценка
□ EV/EBITDA < 8
  Нефтегаз: < 5, IT: допустимо 10–15
□ EV/FCF < 15
```

### Прибыльность (зарабатывает ли?)
```
□ ROE > 15% стабильно 3+ года
□ FCF > 0 и растёт год к году
□ Маржа EBITDA стабильна (не сжимается последние 3 года)
□ Чистая прибыль подтверждена кэшем (OCF > Net Income)
```

### Долг (не слишком ли закредитована?)
```
□ Debt/Equity < 0.8 (для нефинансовых)
□ ND/EBITDA < 3x (< 2x — хорошо, > 3x — опасно)
□ ICR (Interest Coverage Ratio) > 3 — прибыль покрывает проценты
□ Текущая ликвидность (Current Ratio) > 1.5
```

### Дивиденды
```
□ Дивиденды стабильны или растут 5+ лет
□ Payout ratio < 70% (есть запас)
□ Дивиденды обеспечены FCF (FCF > дивидендных выплат)
□ Дивидендная доходность адекватная (> 5% для дивидендной стратегии)
```

## 2. Качество бизнеса (Buffett / Munger / Fisher)

### Понимание бизнеса
```
□ Могу объяснить в 2 предложениях, как компания зарабатывает
□ Понимаю отрасль и её долгосрочные перспективы
□ Знаю основных конкурентов и позицию компании среди них
```

### Конкурентное преимущество (Moat)
```
□ Есть экономический moat. Какой именно:
  - Бренд (Сбербанк, Яндекс)
  - Масштаб / низкие издержки (X5, Магнит)
  - Сетевой эффект (Яндекс, OZON)
  - Switching costs (1С, Позитив)
  - Регуляторный барьер (Транснефть, Мосбиржа)
  - Доступ к ресурсам (Лукойл, Полюс, Алроса)
```

### Менеджмент
```
□ Менеджмент честный и компетентный
□ Менеджмент = крупный акционер (skin in the game)
□ Нет скандалов, вывода активов, размывания
□ Стратегия компании понятна и реалистична
```

### Рост
```
□ Маржи стабильны 3+ года (не сжимаются)
□ Рост органический (не за счёт бесконечных M&A в долг)
□ CAPEX окупается (ROIC > WACC)
□ Дивидендная политика прозрачна и зафиксирована
```

## 3. Скоринговые модели

### Piotroski F-Score
```
□ F-Score ≥ 7 из 9 — финансово здоровая компания
  (Подробный расчёт: @knowledge/advanced_scores.md)

  8–9: сильная → покупать
  5–7: средняя → анализировать детальнее
  0–4: слабая → избегать
```

### Altman Z-Score
```
□ Z-Score > 3.0 — низкий риск банкротства
  Z = 1.2×K1 + 1.4×K2 + 3.3×K3 + 0.6×K4 + K5

  > 3.0: безопасно
  1.8–3.0: серая зона
  < 1.8: высокий риск банкротства → НЕ покупать
```

### Beneish M-Score
```
□ M-Score < -2.22 — отчётность вероятно честная
  > -2.22: подозрение на манипуляции → доп. проверка
```

### Magic Formula (Гринблатт)
```
□ ROIC > 15% — высокая отдача на вложенный капитал
□ Earnings Yield > 10% — дешёвая по прибыли
  EY = EBIT / Enterprise Value
```

### Стратегические формулы
```
□ Graham Number: цена < √(22.5 × EPS × BVPS)
□ Owner Earnings (Баффетт) > 0 и растёт
  OE = Net Income + D&A — CAPEX(поддерж.) ± ΔWC
□ Lynch PEG < 1.0 (для компаний роста)
  PEG = P/E ÷ темп роста прибыли (%)
□ Fisher: min 5 из 6 качественных критериев
```

## 4. Макро и цикл (Marks / Dalio / Templeton)

### Экономический цикл
```
□ Определена текущая фаза цикла:
  Восстановление → Рост → Перегрев → Спад

  Восстановление: покупать циклические (банки, металлы)
  Рост: держать, постепенно фиксировать
  Перегрев: сокращать, переходить в защитные
  Спад: покупать качественные на просадках
```

### Ставка ЦБ
```
□ Текущая ставка и тренд:
  Ставка на пике → акции дёшевы, длинные ОФЗ привлекательны
  Ставка снижается → акции растут, деньги с депозитов → в рынок
  Ставка растёт → осторожность, фокус на дивидендных и защитных

  При ставке > 15%: закредитованные компании под давлением
  (девелоперы, телеком, ритейл с долгом)
```

### Нефть и рубль
```
□ Как нефть влияет на компанию?
  Экспортёры: ↑ нефть = ↑ прибыль
  Импортёры: ↑ нефть → ↑ рубль → ↓ себестоимость

□ Как курс рубля влияет?
  Слабый рубль: плюс для экспортёров, минус для импортёров
  Сургутнефтегаз-п: особый случай (валютная кубышка)
```

### Политический и санкционный риск
```
□ Компания под санкциями? → повышенный дисконт
□ Риск новых санкций? → какой сценарий
□ Импортозаместитель? → долгосрочный попутный ветер
□ Зависимость от импорта → риск
```

### Second-Level Thinking (Маркс)
```
□ Что думает консенсус? → В чём я НЕ согласен с рынком?
□ Сектор не на пике цикла?
□ Хорошая компания ≠ хорошая акция (цена имеет значение)
□ Если все говорят «покупай» → возможно, уже поздно
```

## 5. Риск-управление

### Маржа безопасности
```
□ Маржа безопасности 25–50% от fair value
  Fair value рассчитана минимум 2 методами:
  - DCF (для компаний с предсказуемым FCF)
  - Мультипликаторы (сравнительная оценка)
  - DDM (для дивидендных)
  - NAV (для банков, девелоперов)
```

### Проверка на ловушку стоимости (Value Trap)
```
□ Это НЕ value trap. Проверка:
  - Есть ли структурная причина дешевизны? (отрасль умирает)
  - Прибыль падает 3+ года?
  - Менеджмент разрушает стоимость?
  - Долг растёт, FCF отрицательный?
  - Дивиденды в опасности?

  Если 2+ пунктов «да» → скорее всего ловушка
```

### Размер позиции и выход
```
□ Позиция ≤ 5–10% портфеля (max на одну идею)
□ Готов к просадке -30–50% (горизонт > 3 лет)
□ Есть план выхода:
  - При каких условиях продаю? (тезис сломался)
  - При какой цене фиксирую прибыль?
  - Stop-loss (для спекуляций)
□ Портфель диверсифицирован по 3+ секторам
□ Корреляция с другими позициями < 0.7
```

## 6. Психологическая проверка (против себя)

```
□ Это НЕ покупка из FOMO («все купили, и я хочу»)
□ Это НЕ стадное поведение (проверить: откуда идея?)
□ Применяю second-level thinking (не первую очевидную мысль)
□ Специально искал 3+ аргумента ПРОТИВ покупки
□ Не привязан к «якорной» цене (пересчитал fair value)
□ Готов холдить 3–5+ лет без паники
□ Инвестирую деньги, которые НЕ нужны в ближайшие 1–2 года
□ Не в состоянии эйфории или паники
□ R/R ratio ≥ 1:3 (потенциальная прибыль ≥ 3× потенциальный убыток)
```

## Быстрая оценка (если нет времени на полный чеклист)

### Минимум 5 «да» из 7:
```
1. P/E < 10?
2. ROE > 15%?
3. ND/EBITDA < 2x?
4. FCF > 0?
5. Дивиденды есть и стабильны?
6. Понимаю бизнес?
7. Есть маржа безопасности ≥ 25%?
```

```
5+ «да» → стоит рассмотреть детально (пройти полный чеклист)
3–4 «да» → рискованно, нужен сильный тезис
< 3 «да» → ПРОПУСТИТЬ
```
