# Макроэкономика России для инвестора

> МЕТОДОЛОГИЯ. Конкретные цифры (ставка, курс, нефть) — ВСЕГДА из WebSearch.

## Влияние ключевой ставки ЦБ на активы

```
↑ Ставка ЦБ:
  ↓ Акции (особенно ростовые и закредитованные)
  ↓ Длинные облигации (цена падает)
  ↑ Короткие облигации и депозиты (доходность растёт)
  ↑ Рубль (привлекает carry trade)

↓ Ставка ЦБ:
  ↑ Акции (деньги с депозитов → в акции)
  ↑ Длинные облигации (цена растёт, доходность падает)
  ↓ Депозиты (доходность снижается)
  ↓ Рубль (carry trade уходит)
```

### Секторы: кто выигрывает/проигрывает от высокой ставки
| Сектор | При высокой ставке | Причина |
|--------|-------------------|---------|
| Банки (Сбер, ТКС) | ВЫИГРЫВАЮТ | Чистая процентная маржа растёт |
| Нефть/газ | НЕЙТРАЛЬНО | Низкий долг, зависят от нефти |
| Телеком (МТС) | ПРОИГРЫВАЮТ | Высокий долг, дорогое обслуживание |
| Девелоперы (ПИК, Самолёт) | СИЛЬНО ПРОИГРЫВАЮТ | Ипотека дорожает, спрос падает |
| IT (Яндекс, OZON) | ПРОИГРЫВАЮТ | Ростовые дисконтируются сильнее |
| Ритейл (X5, Магнит) | НЕЙТРАЛЬНО | Стабильный спрос, умеренный долг |
| Металлурги | НЕЙТРАЛЬНО/- | Зависят от спроса, долг невысок |

### Стратегия по циклу ставки
```
Пик ставки:          → покупать длинные ОФЗ + акции (готовиться к снижению)
Начало снижения:     → акции растут, облигации растут
Середина снижения:   → ротация из облигаций в акции
Дно ставки:          → фиксировать прибыль в облигациях, полностью в акциях
Начало повышения:    → сокращать акции, переходить в короткие облигации
```

## Инфляция

### Активы-защитники от инфляции
| Актив | Защита | Почему |
|-------|--------|--------|
| Нефть/газ | Сильная | Экспортная выручка в валюте |
| Золотодобытчики | Сильная | Золото = антиинфляционный актив |
| Сырьевые (удобрения, металлы) | Средняя | Цены привязаны к мировым |
| Банки | Средняя | NIM растёт при высокой инфляции |
| Недвижимость | Средняя | Цены растут, но спрос падает |
| ОФЗ-ИН | Сильная | Номинал индексируется на ИПЦ |

### Проигрывают от инфляции
- Потребительский сектор (маржа сжимается)
- Коммунальные компании (тарифы растут с задержкой)
- Компании с фиксированными ценами/контрактами

## Нефть и рубль

### Бюджетное правило
```
Сверхдоходы (Urals > отсечка): → в ФНБ
Недостаток (Urals < отсечка):  → продажа валюты из ФНБ
Актуальную цену отсечки проверяй через WebSearch
```

### Корреляции
```
↑ Нефть → ↑ Рубль (больше валютной выручки)
↑ Нефть → ↑ Нефтяники (Лукойл, Роснефть, Татнефть)
↑ Нефть → ↑ Бюджет → ↑ Расходы → ↑ Инфляция

↓ Нефть → ↓ Рубль → ↑ Экспортёры (в рублях)
                    → ↓ Импортёры
```

### Нефтяные компании РФ: чувствительность
| Компания | Чувствительность к нефти | Комментарий |
|----------|------------------------|-------------|
| Роснефть | Высокая | Крупнейший добытчик |
| Лукойл | Высокая | Низкий долг, высокие дивиденды |
| Газпром нефть | Высокая | Дочка Газпрома |
| Татнефть | Средняя | Диверсификация (шинный бизнес) |
| Сургутнефтегаз | Особая | Кубышка в валюте: ↓ рубль → ↑ прибыль |
| Новатэк | Средняя | СПГ = отдельная ценовая динамика |

## Санкции — структурное влияние

```
Сильно пострадали:
- Газпром (потеря европейского рынка)
- Авиакомпании (нет запчастей, лизинг)
- Банки под SDN (заморожены резервы)

Выиграли от импортозамещения:
- IT (Яндекс, Позитив, Астра) — уход западных конкурентов
- Ритейл — замещение брендов
- Агро (Русагро, Фосагро) — продовольственная безопасность
```

## Курс рубля — факторы

```
Укрепляют рубль:              Ослабляют рубль:
+ Высокая ставка ЦБ           - Падение нефти
+ Продажа валюты из ФНБ       - Отток капитала
+ Высокие цены на нефть       - Рост импорта
+ Обязательная продажа        - Бюджетный дефицит
  валютной выручки            - Снижение ставки ЦБ
+ Сокращение импорта          - Геополитические шоки
```

### Бенефициары слабого рубля
- Экспортёры: выручка в валюте, расходы в рублях (Лукойл, Фосагро)
- Сургутнефтегаз: валютная кубышка
- Золотодобытчики: Полюс (золото в $ → прибыль в ₽)

### Пострадавшие от слабого рубля
- Импортёры: себестоимость растёт
- Компании с валютным долгом

## Когда дивидендные лучше ростовых

### Правило переключения
```
Ставка ЦБ > 12%:  Фокус на дивидендные + облигации
Ставка ЦБ 8–12%:  Баланс дивидендных и ростовых
Ставка ЦБ < 8%:   Фокус на ростовые

Формула: если div yield > безрисковая ставка × 0.7 — акция интересна
```
