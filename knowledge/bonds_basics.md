# Облигации для инвестора

## Типы облигаций на российском рынке

### ОФЗ — облигации федерального займа (государственные)
Эмитент: Минфин РФ. Считаются безрисковыми (дефолт маловероятен).

| Тип | Код | Купон | Для кого |
|-----|-----|-------|----------|
| ОФЗ-ПД | 26ХХХ | Постоянный (фиксированный) | Ставка на снижение ставки ЦБ |
| ОФЗ-ПК (флоатеры) | 29ХХХ | Плавающий (привязан к RUONIA) | Защита от роста ставки |
| ОФЗ-ИН (линкеры) | 52ХХХ | Фиксированный + индексация номинала на ИПЦ | Защита от инфляции |
| ОФЗ-АД | 46ХХХ | Постоянный, амортизация номинала | Постепенный возврат тела |
| ОФЗ-н (народные) | 53ХХХ | Растущий | Для начинающих |

### Корпоративные облигации
```
Первый эшелон (AAA–AA):
  - Сбербанк, Лукойл, Газпром, РЖД
  - Доходность: ОФЗ + 0.5–1.5%
  - Риск: минимальный

Второй эшелон (A–BBB):
  - Средние компании
  - Доходность: ОФЗ + 2–4%
  - Риск: умеренный

ВДО — высокодоходные облигации (BB и ниже):
  - Малый и средний бизнес
  - Доходность: ОФЗ + 5–10%
  - Риск: ВЫСОКИЙ (дефолты 3–7% в год)
```

### Муниципальные облигации
- Эмитенты: регионы и города (Москва, Самарская обл.)
- Доходность: между ОФЗ и корпоративными
- Ликвидность: низкая

## Дюрация (Duration)

### Что это
Мера чувствительности цены облигации к изменению процентных ставок. Измеряется в годах.

### Правило
```
Дюрация = 6 → при росте ставки на 1% цена облигации упадёт примерно на 6%
Дюрация = 2 → при росте ставки на 1% цена упадёт примерно на 2%

Точная формула:
ΔP ≈ -D × Δr × P

Пример:
Облигация: номинал 1000 руб, дюрация 7
Ставка выросла на 2%
ΔP ≈ -7 × 0.02 × 1000 = -140 руб
Новая цена ≈ 860 руб (падение 14%)
```

### Модифицированная дюрация
```
MD = D / (1 + YTM/n)

D = дюрация Маколея
YTM = доходность к погашению
n = количество купонных выплат в году

MD точнее показывает чувствительность к ставке
```

### Как использовать
```
Ожидаете снижение ставки → покупайте длинные облигации (высокая дюрация)
  Причина: при снижении ставки цена вырастет сильнее

Ожидаете рост ставки → покупайте короткие облигации (низкая дюрация)
  Причина: меньше потеряете при росте ставки

Не уверены → покупайте флоатеры (ОФЗ-ПК)
  Причина: купон подстраивается под ставку
```

## YTM — доходность к погашению

### Что это
Полная годовая доходность, если:
1. Держать облигацию до погашения
2. Реинвестировать все купоны под ту же ставку

### Из чего складывается
```
YTM = Купонная доходность + Доход/убыток от цены

Пример 1 (облигация ниже номинала):
Купон: 80 руб/год (8%)
Цена покупки: 900 руб (90% от номинала)
До погашения: 3 года
Доход от цены: (1000 - 900) / 3 = 33 руб/год

Примерный YTM ≈ (80 + 33) / 950 ≈ 11.9%

Пример 2 (облигация выше номинала):
Купон: 120 руб/год (12%)
Цена покупки: 1050 руб (105% от номинала)
До погашения: 2 года
Убыток от цены: (1000 - 1050) / 2 = -25 руб/год

Примерный YTM ≈ (120 - 25) / 1025 ≈ 9.3%
```

### Эффективная доходность
```
Эффективная YTM учитывает реинвестирование купонов.
Чем чаще купоны — тем выше эффективная доходность (эффект сложного процента).

Купон 2 раза в год > Купон 1 раз в год (при одинаковой ставке)
```

## Когда облигации лучше акций

### Правило сравнения
```
Облигации предпочтительнее, когда:
1. YTM облигаций > дивидендной доходности акций + ожидаемый рост
2. Ставка ЦБ на пике или близко (потенциал роста цены)
3. Рынок акций перегрет (P/E > исторических средних)
4. Высокая неопределённость (геополитика, санкции)
5. Горизонт инвестирования < 2 лет

Акции предпочтительнее, когда:
1. Ставка ЦБ снижается (переток из облигаций в акции)
2. P/E рынка ниже исторических средних
3. Инфляция ускоряется (акции = реальные активы)
4. Горизонт > 3 лет
```

### Текущая ситуация (2025–2026)
```
Ставка ЦБ: 21% (пик)
YTM ОФЗ 10 лет: ~15%
YTM корпоративных (1 эшелон): ~17–18%
Средняя див доходность IMOEX: ~9–10%

Вывод: облигации дают доходность сопоставимую с акциями,
       но с меньшим риском.
       При снижении ставки — длинные ОФЗ дадут рост цены +20–30%.
```

## Стратегия: ставка на снижение ставки ЦБ

### Логика
```
Ставка ЦБ на пике (21%) → начнёт снижаться в 2025–2026
→ Доходности облигаций упадут
→ Цены облигаций вырастут
→ Чем длиннее облигация (выше дюрация) — тем больше рост цены
```

### Расчёт потенциала
```
ОФЗ 26244 (погашение 2034):
Дюрация: ~7
Текущая YTM: 15.5%
Если YTM снизится до 10% (ставка ЦБ = 12%):
Рост цены ≈ 7 × 5.5% ≈ 38.5%
+ Купон за год: 11.25%
Итого за 1–2 года: ~50% потенциальная доходность

ОФЗ 26238 (погашение 2041):
Дюрация: ~10
Если YTM снизится с 15% до 10%:
Рост цены ≈ 10 × 5% ≈ 50%
+ Купон: 7.15%
Итого: ~57% потенциал
```

### Риски стратегии
```
❌ Ставку не снизят (инфляция останется высокой)
❌ Снижение будет медленным (2–3 года вместо 1)
❌ Кредитный риск (для корпоративных)
❌ Ликвидность (для длинных выпусков)
```

## Лестница облигаций (Bond Ladder)

### Суть
Распределить деньги равномерно по срокам погашения.

```
Пример лестницы на 1 000 000 руб:
200 000 → ОФЗ с погашением через 1 год
200 000 → ОФЗ с погашением через 2 года
200 000 → ОФЗ с погашением через 3 года
200 000 → ОФЗ с погашением через 5 лет
200 000 → ОФЗ с погашением через 7 лет

При погашении ближней → реинвестировать в длинную
```

### Преимущества
- Снижает процентный риск (усреднение)
- Регулярный кэш (погашения каждый год)
- Не нужно угадывать момент

## Налоги на облигации

### Купонный доход
```
ОФЗ: НДФЛ 13–15% с купонов (с 2021 года)
Корпоративные: НДФЛ 13–15% с купонов

На ИИС тип Б: освобождение от налога на купоны (кроме ОФЗ-н)
```

### Доход от роста цены
```
Продал дороже купил → НДФЛ 13–15% с разницы
ЛДВ (3 года): освобождение до 3 млн руб × кол-во лет

НКД (накопленный купонный доход):
  При покупке: платишь НКД продавцу
  При продаже: получаешь НКД от покупателя
  Налогом облагается разница (полученный - уплаченный)
```

## Замещающие облигации

### Что это
Замещающие облигации — это рублёвые/валютные облигации, выпущенные вместо заблокированных еврооблигаций. Эмитент заменил ("заместил") старый выпуск в Euroclear на новый в НРД.

### Параметры
```
Валюта: USD, EUR, иногда CNY (привязка номинала к валюте, расчёты в рублях по курсу)
Доходность: 5–8% в валюте (выше, чем ОФЗ в рублях с учётом валютного риска)
Эмитенты: Газпром, Лукойл, ГМК Норникель, РЖД, ГТЛК, Совкомфлот
Ликвидность: средняя (ниже ОФЗ, выше корпоративных)
```

### Когда покупать
- Ставка на ослабление рубля (если рубль ослабнет → рублёвый доход вырастет)
- Как хедж от девальвации в портфеле
- Альтернатива валютным вкладам (ставки по USD-вкладам ~1–2%, замещающие ~5–7%)

### Риски
- Кредитный риск эмитента
- Валютный риск (рубль укрепится → убыток в рублях)
- Низкая ликвидность некоторых выпусков

## Carry Trade на рубле

### Суть
Занять валюту с низкой ставкой (USD ~5%, EUR ~4%) → конвертировать в рубли → разместить под высокую ставку (ОФЗ ~15%, депозит ~20%).

### Параметры
```
Доходность: спред 10–15% годовых (минус комиссии и конвертация)
Риск: резкая девальвация рубля может уничтожить весь спред за дни
  - 2014: рубль -60% за 3 мес → carry trade убыточен
  - 2022: рубль -40% за 2 недели → carry trade убыточен
  - 2023–2025: рубль стабилен → carry trade прибылен
```

### Влияние на рынок
```
Приток carry trade → укрепление рубля → поддержка рынка акций
Разворот carry trade (снижение ставки или геориск) → ослабление рубля → давление на рынок
```

**Для инвестора:** carry trade создаёт "пол" для рубля пока ставка высокая, но это хрупкая поддержка. При снижении ставки ЦБ carry trade будет сворачиваться → давление на рубль.

## Чек-лист перед покупкой облигации

```
✅ Кредитный рейтинг эмитента (не ниже BBB для консервативных)
✅ YTM адекватная (сравнить с ОФЗ аналогичного срока)
✅ Дюрация соответствует ожиданиям по ставке
✅ Ликвидность (спред bid-ask < 0.5%, оборот > 1 млн/день)
✅ Оферта (если есть — по какой цене и когда)
✅ Амортизация (если есть — как влияет на дюрацию)
✅ Ковенанты (ограничения для эмитента — чем строже, тем лучше)
✅ Финансовое состояние эмитента (ND/EBITDA < 3, ICR > 3)
```
