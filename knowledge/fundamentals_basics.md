# Основы фундаментального анализа и отчетность

> Урок №2 — Отчетность

## Голубые фишки (Blue Chips)

Акции крупнейших, наиболее ликвидных и надёжных компаний. На MOEX:
- Высокая ликвидность (оборот > 1 млрд руб/день)
- Стабильный бизнес, история 10+ лет
- Входят в индекс IMOEX
- Примеры: Сбербанк, Лукойл, Газпром, Роснефть, Новатэк, Яндекс, Норникель, Татнефть

Для новичков — начинать с голубых фишек (группы 1-2 из базы данных).

---

## Короткая продажа (шорт)

Продажа акций которых у тебя нет (берёшь в долг у брокера → продаёшь → ждёшь падения → откупаешь дешевле → возвращаешь).

```
Заработок: цена упала (продал дорого, купил дёшево)
Убыток: цена выросла (убыток неограничен!)
Комиссия: ежедневная плата за заём акций

Риски:
- Убыток НЕОГРАНИЧЕН (акция может расти бесконечно)
- Маржин-колл при росте цены
- Принудительное закрытие брокером
- Дивиденды: шортист ПЛАТИТ дивиденд владельцу

Правило: долгосрочный инвестор НЕ шортит. Это инструмент для профессионалов.
```

## Цель фундаментального анализа

Определить привлекательность компании для инвестиций на **среднесрочную перспективу** (от 1 года).

## Что такое стоимость компании (для публичных компаний)

- **Капитализация** = стоимость акции * количество акций
- **Enterprise Value (EV)**, стоимость бизнеса = капитализация + долг - кэш

## Когда компания привлекательна по фундаменталу

1. Диапазон справедливой цены > текущей цены
2. *Катализаторы в будущем* (возврат к выплате дивидендов, хороший отчет, сделка M&A — выгодная для компании и другие) — спекуляции по фундаменталу

## Материал для фундаментального анализа

- Отчетность
- Интервью
- Презентации
- Пояснения
- Конференц-звонки

### Где смотреть

1. Сайты компаний
2. e-disclosure — https://www.e-disclosure.ru/

## Виды отчетности (в РФ)

- **Производственная** отчетность
- **Финансовая** отчетность — делится на стандарты:
  - **МСФО** (Международные стандарты финансовой отчетности)
  - **РСБУ** (Российские стандарты бухгалтерского учета)

## МСФО vs РСБУ — детальное сравнение

| Критерий | МСФО | РСБУ |
|----------|------|------|
| **Назначение** | Для инвесторов и рынка | Для налоговой и регуляторов |
| **Консолидация** | ДА — все дочерние компании в одном отчёте | НЕТ — только юрлицо, без дочек |
| **Оценка активов** | Справедливая стоимость (рыночная) | Историческая стоимость (цена покупки) |
| **Обязательность** | Для всех публичных компаний на MOEX | Для всех юрлиц в РФ |
| **Периодичность** | Полугодовая + годовая | Квартальная + годовая |
| **Аудит** | Обязательный (Big 4 или крупные) | Обязательный |
| **Язык** | Русский + английский | Только русский |
| **Где смотреть** | e-disclosure.ru, сайт компании | e-disclosure.ru, ФНС |

### Когда МСФО и РСБУ расходятся — красные флаги

```
Нормально:
  МСФО прибыль > РСБУ — консолидация дочек (Россети, Мечел)
  РСБУ прибыль > МСФО — дивиденды от дочек (холдинги)

Подозрительно:
  Огромная разница (>50%) без очевидной причины → проверить примечания
  РСБУ показывает прибыль, МСФО — убыток → дочки генерируют убытки
  Резкое изменение разницы год к году → переоценка или манипуляции
```

### Для инвестора: ВСЕГДА используй МСФО
- Показывает реальную картину бизнеса (с дочками)
- Активы по рыночной стоимости (не по цене 20 лет назад)
- Международно сравнимая (можно сравнить с зарубежными аналогами)
- **Исключение:** если компания не публикует МСФО (малые компании) — РСБУ как запасной вариант

## Ключевой принцип

Чем больше вы найдете информации по компании, тем лучше. Смотреть надо интервью, презентации, отчеты (финансовые и производственные), сайт компании.
