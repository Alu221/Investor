# Бухгалтерский баланс

> Урок №4 — Бухгалтерский баланс

## Общие принципы

- Бухгалтерский баланс = **отчет на дату**
- Баланс компании состоит из **имущества (активов)** и **источников его финансирования (пассивов)**

## Структура активов

- **Оборотные активы** — до 1 года
- **Внеоборотные активы** — более 1 года

## Структура пассивов

- **Краткосрочные обязательства** — до 1 года
- **Долгосрочные обязательства** — более 1 года

## Зависимость от отрасли

Структура Активов и Пассивов сильно зависит от отрасли:
- **Производственные компании** — основной актив: основные средства (здания/сооружения)
- **IT-компании** — основной актив: нематериальные активы (программное обеспечение, лицензии, патенты и т.д.)

## Капитал

**Капитал** — это средства акционеров компании.

## Главное тождество

```
Активы = Обязательства + Капитал
```

### Примеры операций

- Компания купила актив: +основные средства, -кэш (или +долг)
- Компания выплатила дивиденд: -нераспределенная прибыль, -кэш

---

## Специфика баланса банков

Баланс банка принципиально отличается от промышленной компании.

### Структура активов банка
```
Активы банка:
  Кредиты клиентам (60-70%) — основной актив
  Ценные бумаги (10-20%) — портфель ОФЗ, корп. облигации
  Денежные средства (5-10%)
  Межбанковские кредиты (5-10%)
  Основные средства (<5%) — здания, IT
```

### Структура пассивов банка
```
Пассивы банка:
  Депозиты клиентов (60-70%) — основной источник
  Межбанковские займы (10-15%)
  Выпущенные облигации (5-10%)
  Собственный капитал (8-15%) — очень маленькая доля!
```

### Ключевые метрики банка (НЕ используй EV/EBITDA!)

| Метрика | Формула | Норма | Комментарий |
|---------|---------|-------|-------------|
| **Н1.0** (достаточность капитала) | Капитал / Активы (взвешенные по риску) | > 12% (ЦБ требует > 8%) | Ниже 8% = отзыв лицензии |
| **NPL ratio** (просроченные кредиты) | Просроченные кредиты / Кредитный портфель | < 5% — хорошо, > 10% — проблемы | Качество кредитного портфеля |
| **NIM** (чистая процентная маржа) | (% доходы - % расходы) / Активы | 3-5% для РФ банков | Растёт при высокой ставке ЦБ |
| **Cost-to-Income** (CIR) | Операционные расходы / Операционные доходы | < 40% — отлично, > 60% — плохо | Эффективность банка |
| **ROE** | Чистая прибыль / Капитал | > 15% для банков | Главная метрика эффективности |
| **P/BV** | Капитализация / Собственный капитал | 0.7-1.5 для РФ | Ключевой мультипликатор банка |

### Почему банки оцениваются иначе
```
НЕ применять к банкам:
  ❌ EV/EBITDA — у банков нет EBITDA в классическом смысле
  ❌ FCF — денежные потоки банка = кредиты/депозиты, не операционный поток
  ❌ ND/EBITDA — долг банка = депозиты, это его бизнес

ПРИМЕНЯТЬ к банкам:
  ✅ P/E — отношение капитализации к прибыли
  ✅ P/BV — ключевой мультипликатор (дёшево < 1, дорого > 2)
  ✅ ROE — эффективность использования капитала
  ✅ Дивидендная доходность

Правило: Высокий ROE + низкий P/BV = недооценённый банк
Пример: Сбер 2022 — P/BV 0.5 при ROE 24% → вырос в 3 раза
```

### Риски баланса банка
```
Кредитный риск: NPL растёт → резервы растут → прибыль падает
Процентный риск: ставка ЦБ растёт → NIM может расти (если кредиты пересчитываются быстрее депозитов)
Ликвидность: паника вкладчиков → отток депозитов → кризис ликвидности
Достаточность капитала: убытки → капитал падает → Н1 ниже нормы → ограничения ЦБ
```
