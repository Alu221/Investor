# Управление портфелем

## Диверсификация

### По секторам
Не более 25–30% портфеля в одном секторе. Минимум 3–4 сектора.

```
Рекомендуемое распределение:
Нефть и газ:       15–25%
Банки/финансы:     15–20%
IT/Технологии:     10–15%
Металлы/Добыча:    10–15%
Ритейл/Потреб:     10–15%
Телеком:           5–10%
Электроэнергетика: 5–10%
```

### По классам активов
```
Акции:       40–70% (зависит от профиля риска)
Облигации:   20–40%
Кэш/депозит: 5–15% (сухой порох для просадок)
Золото:      5–10% (хедж от инфляции и георисков)
```

### Корреляция
- Цель: корреляция между позициями < 0.7
- Примеры низкой корреляции в РФ:
  - Полюс (золото) и Сбербанк (банки) — корреляция ~0.3
  - МТС (телеком) и Лукойл (нефть) — корреляция ~0.4
- Примеры высокой корреляции (избегать перекоса):
  - Лукойл и Роснефть — корреляция ~0.85
  - Сбербанк и ТКС — корреляция ~0.75

## Ребалансировка

### Когда ребалансировать
1. **По времени**: ежеквартально (раз в 3 месяца)
2. **По отклонению**: когда позиция отклонилась > 5% от целевого веса
3. **По событию**: после значимого корпоративного события

### Как ребалансировать
```
Шаг 1: Сравнить текущие веса с целевыми
Шаг 2: Продать то, что выросло выше целевого
Шаг 3: Купить то, что упало ниже целевого
Шаг 4: Учесть налоги (продажа = налог на прибыль)

Пример:
Целевой вес Лукойла: 15%
Текущий вес: 22% (сильно вырос)
Действие: продать часть, довести до 15–17%
```

### Налоговая оптимизация при ребалансировке
- Использовать ИИС тип Б (без налога на прибыль)
- Использовать льготу долгосрочного владения (3+ года)
- Сначала продавать позиции в убытке (tax-loss harvesting)

## Risk/Reward Ratio

### Минимальное соотношение: 1:3
```
Потенциальный убыток: 10%
Потенциальная прибыль: минимум 30%

Если R/R < 1:2 — сделка не стоит риска
Если R/R > 1:5 — перепроверить допущения (может быть ловушка)
```

### Расчёт
```
R/R = (Целевая цена - Текущая цена) / (Текущая цена - Stop-loss)

Пример:
Текущая цена: 100 руб
Целевая цена: 140 руб (апсайд 40%)
Stop-loss: 87 руб (даунсайд 13%)

R/R = (140 - 100) / (100 - 87) = 40 / 13 = 3.1 — ХОРОШО
```

## Position Sizing (размер позиции)

### Правила
```
Максимум на одну идею:           10% портфеля
Стандартная позиция:             3–5% портфеля
Высокая уверенность (5/5):      7–10% портфеля
Средняя уверенность (3/5):      3–5% портфеля
Низкая уверенность (1–2/5):     1–3% портфеля
Спекулятивная идея:              1–2% портфеля
```

### Формула Келли (упрощённая)
```
f = (p × b - q) / b

f = доля капитала на ставку
p = вероятность успеха
q = 1 - p (вероятность неудачи)
b = отношение выигрыша к проигрышу (R/R)

Пример:
p = 60% (вероятность роста)
b = 3 (R/R = 1:3)
q = 40%

f = (0.6 × 3 - 0.4) / 3 = 1.4 / 3 = 0.47 = 47%

На практике: использовать 1/4 Келли = 12% — более консервативно
```

### Правило 2% риска
```
Максимальный убыток на одну сделку = 2% от портфеля

Если портфель = 1 000 000 руб
Макс убыток на сделку = 20 000 руб
Stop-loss = -10%
→ Максимальная позиция = 200 000 руб (20% портфеля)

Если stop-loss = -5%
→ Максимальная позиция = 400 000 руб (40% портфеля, но ограничиваем 10%)
```

## Stop-Loss

### Для спекуляций (обязателен)
```
Жёсткий stop-loss: -7% до -10% от цены входа
Trailing stop: подтягивается за ценой (например, -10% от максимума)
По волатильности: 2 × ATR(14) ниже цены входа
```

### Для долгосрочных инвестиций (необязателен)
```
Вместо stop-loss — пересмотр тезиса:
1. Тезис не изменился → держать, можно докупить
2. Тезис сломался → продавать независимо от цены

Примеры сломанного тезиса:
- Компания отменила дивиденды (покупали ради дивидендов)
- Резкое ухудшение финансов (долг вырос в 3 раза)
- Потеря конкурентного преимущества
- Смена менеджмента на худший
```

## 3 модели портфелей

### 1. Консервативный портфель
```
Цель: сохранение капитала + доход 15–18% годовых
Горизонт: 1–3 года
Риск: низкий

Структура:
ОФЗ (длинные + флоатеры):     40%
Корпоративные облигации:       15%
Дивидендные акции:             30%
  - Лукойл:        8%
  - Сбербанк:      7%
  - Татнефть:      5%
  - МТС:           5%
  - Транснефть:    5%
Золото (фонд/Полюс):          10%
Кэш:                           5%

Ожидаемая доходность: 15–18%
Макс просадка: -10–15%
```

### 2. Умеренный портфель
```
Цель: рост капитала 20–25% годовых
Горизонт: 3–5 лет
Риск: средний

Структура:
Акции (голубые фишки):         55%
  - Сбербанк:      10%
  - Лукойл:        8%
  - Яндекс:        7%
  - Полюс:         5%
  - Татнефть:      5%
  - Х5:            5%
  - Новатэк:       5%
  - ТКС:           5%
  - Позитив:       5%
Облигации (ОФЗ + корп):       25%
Золото:                        10%
Кэш:                          10%

Ожидаемая доходность: 20–25%
Макс просадка: -20–30%
```

### 3. Агрессивный портфель
```
Цель: максимальный рост 30%+ годовых
Горизонт: 5+ лет
Риск: высокий

Структура:
Акции роста:                   50%
  - Яндекс:        10%
  - OZON:           8%
  - Позитив:        8%
  - Астра:          7%
  - ТКС:           7%
  - HeadHunter:     5%
  - Whoosh:         5%
Дивидендные акции:             25%
  - Лукойл:        7%
  - Сбербанк:      7%
  - Полюс:         6%
  - Татнефть:      5%
Облигации (короткие ВДО):     10%
Кэш:                          15%

Ожидаемая доходность: 25–35%
Макс просадка: -30–50%
```

## Метрики портфеля

### Что отслеживать
| Метрика | Формула | Целевое значение |
|---------|---------|-----------------|
| Доходность | (Стоимость_конец - Стоимость_начало) / Стоимость_начало | > Индекс МосБиржи |
| Sharpe Ratio | (Rp - Rf) / σp | > 1.0 |
| Макс просадка | Макс падение от пика до дна | < -30% |
| Beta портфеля | Средневзвешенная β позиций | 0.8–1.2 |
| Див доходность | Годовые дивиденды / Стоимость портфеля | > 5% |
| Концентрация | Вес топ-3 позиций | < 35% |
