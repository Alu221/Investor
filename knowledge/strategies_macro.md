## Ray Dalio — All Weather и Risk Parity

### All Weather Portfolio (всепогодный портфель)

```
Распределение:
  30%  — Акции (S&P 500 / широкий рынок)
  40%  — Длинные гос. облигации (20-25 лет)
  15%  — Среднесрочные гос. облигации (7-10 лет)
   7.5% — Золото
   7.5% — Коммодитиз (нефть, металлы, сельхоз)

Ребалансировка: 1 раз в год (или при отклонении > 5%)
```

### Логика 4 сезонов экономики

| Сезон | Условия | Что растёт |
|-------|---------|-----------|
| **Весна** | Рост ↑, Инфляция ↓ | Акции, корп. облигации |
| **Лето** | Рост ↑, Инфляция ↑ | Коммодитиз, TIPS, акции |
| **Осень** | Рост ↓, Инфляция ↓ | Длинные гос. облигации, акции |
| **Зима** | Рост ↓, Инфляция ↑ | Золото, коммодитиз, TIPS |

### Risk Parity (паритет риска)

```
Идея: каждый актив вносит ОДИНАКОВЫЙ вклад в риск портфеля

Вес актива = (1 / σ_актива) / Σ(1 / σ_всех)

где σ — стандартное отклонение (волатильность) за 12 мес

Пример:
  Акции σ = 15%  → вес = (1/15) / [(1/15) + (1/7) + (1/20)]
  Облиги σ = 7%  → вес = (1/7) / [сумма]
  Золото σ = 20% → вес = (1/20) / [сумма]

Результат: облигации получают больший вес (менее волатильны)
```

### 5 правил Далио

1. **5 некоррелированных потоков дохода** — корреляция < 0.3 между ними
2. **Диверсификация — единственный бесплатный обед** в инвестициях
3. **Не ставь на прогнозы** — готовься ко всем сценариям
4. **Делеверидж** — снижай долговую нагрузку в неопределённости
5. **Принципы > Эмоции** — систематический подход побеждает интуицию

### Debt Cycle (долговой цикл Далио)

```
Краткосрочный цикл: 5-8 лет
  Расширение кредита → Рост → Инфляция →
  Ужесточение ставок → Замедление → Рецессия → Снижение ставок

Долгосрочный цикл: 50-75 лет
  Накопление долга десятилетиями → Делеверидж (болезненный) →
  3 варианта: дефолт / инфляция / реструктуризация

Индикатор: Debt/GDP > 250% = поздняя стадия долгового цикла
```

## George Soros — рефлексивность и макро-трейдинг

### Теория рефлексивности

```
Цикл бум-бюст (boom-bust):

1. Зарождение тренда (непризнанный)
2. Самоподкрепление (тренд + bias усиливают друг друга)
3. Успешное тестирование (коррекция, тренд выдерживает)
4. Растущая уверенность (отрыв восприятия от реальности)
5. Точка расхождения (реальность ≠ ожиданиям)
6. Сумерки (все ещё верят, но фундамент рушится)
7. Крах (быстрый разворот, паника)

Ключ: момент когда восприятие МАКСИМАЛЬНО расходится с реальностью =
      точка входа (шорт) или выхода
```

### Принципы Сороса

| Принцип | Описание |
|---------|----------|
| **Фаллибилизм** | Все наши модели мира неполны и ошибочны |
| **Рефлексивность** | Участники рынка ВЛИЯЮТ на фундаментал (самосбывающийся прогноз) |
| **Асимметрия** | Ставь мало, когда не уверен. Ставь ОЧЕНЬ много, когда уверен |
| **Выживание** | Сначала выживи, потом зарабатывай. Управляй риском |
| **Гипотезы** | Инвестиционный тезис — это гипотеза. Данные против? Выходи |

### Макро-трейдинг Сороса

```
Алгоритм:
1. Сформулируй макро-тезис (например: "ЦБ будет вынужден девальвировать")
2. Найди рынок с максимальным расхождением ожиданий и реальности
3. Определи катализатор (событие, которое вскроет расхождение)
4. Открой позицию с определённым стоп-лоссом
5. Если рынок подтверждает тезис — наращивай позицию агрессивно
6. Если опровергает — выходи немедленно, без эго
```

## John Templeton — контрарианство и глобализация

### Контрарианский подход

> "Покупай в момент максимального пессимизма" — Templeton

```
Индикаторы максимального пессимизма:
  - VIX > 40 (экстремальный страх)
  - Put/Call Ratio > 1.2
  - % быков (AAII) < 20%
  - Газеты пишут о "конце рынка"
  - Фонды фиксируют рекордные оттоки
  - P/E рынка < 10 (для развитых рынков)

Покупать: когда ВСЕ вышеперечисленное совпадает
```

### Критерии Темплтона

| Параметр | Порог |
|----------|-------|
| P/E | < 8-10 на дне (для рынка в целом) |
| P/E акции | < 5 при наличии бизнеса и прибыли |
| Горизонт | 4-5 лет (среднесрочный контрарианец) |
| Диверсификация | Глобальная (10+ стран) |
| Продажа | Когда найдена более дешёвая альтернатива |

### Глобальная диверсификация

```
Правило Темплтона:
  - Ищи самый дешёвый рынок в МИРЕ
  - Не привязывайся к домашнему рынку (home bias = ошибка)
  - Развивающиеся рынки дают лучшие возможности (и больший риск)

Метрика: CAPE (Shiller P/E) по странам
  CAPE < 10 — рынок экстремально дёшев (покупать)
  CAPE 10-15 — дёшев
  CAPE 15-25 — справедливо
  CAPE > 25 — дорог
  CAPE > 35 — экстремально дорог (сокращать)
```

## Адаптация макро-стратегий для российского рынка

### All Weather для РФ

```
Адаптированный портфель:
  25%  — Акции (IMOEX / голубые фишки)
  30%  — ОФЗ длинные (10-15 лет)
  15%  — ОФЗ короткие (2-3 года) / фонды денежного рынка
  15%  — Золото (GLDRUB / физическое / фонд)
  10%  — Замещающие еврооблигации / юаневые облигации
   5%  — Коммодитиз (через акции: нефтянка, металлы)

Особенности:
  - Нет прямого доступа к TIPS → замена: ОФЗ-ИН (индексируемые на инфляцию)
  - Ограниченный доступ к глобальным рынкам → концентрация на РФ + дружественные рынки
  - Высокая инфляция → больше доля золота и реальных активов
```

### Российские макро-индикаторы

| Индикатор | Что смотреть | Где |
|-----------|-------------|-----|
| Ключевая ставка ЦБ РФ | Направление и темп изменения | cbr.ru |
| ИПЦ (инфляция) | > 8% = высокая, < 4% = таргет | rosstat.gov.ru |
| Нефть Brent | > $80 = позитив для РФ, < $50 = риск | финансовые терминалы |
| USD/RUB | Тренд ослабления = инфляция, золото | moex.com |
| Кривая ОФЗ | Инверсия = ожидание рецессии | rusbonds.ru |
| PMI Manufacturing | > 50 = рост, < 50 = сжатие | S&P Global / IHS Markit |

### Когда покупать российский рынок (контрарианский)

```
Сигналы дна:
  - IMOEX P/E < 5 (экстремально дёшево по мировым стандартам)
  - Дивидендная доходность индекса > 10%
  - Нерезиденты массово выходят
  - СМИ пишут "инвестировать в Россию безумие"
  - ЦБ начинает цикл снижения ставки

Сигналы вершины:
  - P/E > 12-15 (для РФ это дорого)
  - Все говорят "Россия — новый инвестиционный рай"
  - Массовый приток розничных инвесторов-новичков
  - ЦБ начинает цикл повышения ставки
```
